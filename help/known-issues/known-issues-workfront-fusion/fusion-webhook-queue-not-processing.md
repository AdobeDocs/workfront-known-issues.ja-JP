# Workfront Fusion：web フックキューが処理されない

Workfront Fusion で web フックシナリオをトリガーするイベントが発生すると、シナリオは実行されず、イベントは web フックのキューに残ります。

_最初に報告されたのは 2024年9月2日（PT）です。_
