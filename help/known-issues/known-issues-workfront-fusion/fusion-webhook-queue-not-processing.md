# Workfront Fusion:Webhook キューが処理されない

Workfront Fusion で Webhook シナリオをトリガーにするイベントが発生すると、シナリオは実行されず、イベントは Webhook のキューに残ります。

_最初に報告されたのは 2024年9月2日（PT）です。_
