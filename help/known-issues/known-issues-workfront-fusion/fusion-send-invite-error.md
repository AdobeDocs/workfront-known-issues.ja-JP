# [!DNL Workfront Fusion]：[!DNL Workfront]／[!UICONTROL その他のアクション]モジュールで招待を送信する際にエラーが発生する

>[!NOTE]
>
>この問題は解決されました。

「[!UICONTROL 「user」が null なので、「com.attask.biz.User.getLicenseType()」を呼び出すことができません]」

_最初に報告されたのは 2023年9月14日（PT）です。_
