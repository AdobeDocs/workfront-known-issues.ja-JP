# Workfront Fusion：実行の詳細が処理中として表示される

ユーザーが実行の詳細を表示しようとすると、実行が完了してから長い間、実行に「処理中」バッジが表示されます。

ユーザーがシナリオの詳細を表示しようとすると、「ログの詳細が存在しません」というメッセージが表示されます。

_最初に報告されたのは 2024年8月22日（PT）です。_
