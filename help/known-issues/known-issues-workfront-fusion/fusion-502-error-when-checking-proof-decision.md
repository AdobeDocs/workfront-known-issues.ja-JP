# [!DNL Workfront Fusion]：シナリオに [!DNL Workfront] でプルーフ決定を確認するモジュールが含まれる場合の 502 エラー。

>[!NOTE]
>
>この問題は 2022年8月22日に修正されました。

[!DNL Workfront] で行われたプルーフ決定を確認するモジュールを含むシナリオを実行すると、そのモジュールは 502 エラーを返します。

_最初に報告されたのは 2022年7月28日です。_
