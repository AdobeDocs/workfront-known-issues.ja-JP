# Workfront Fusion：「今後」の設定を開始すると、すべてのデータが処理される

シナリオの「開始する日時を選択」設定として「今後」を選択した後、ユーザーがシナリオを以前のバージョンに戻すと、開始オプションが選択された後のデータのみが含まれるのではなく、日付に関係なくすべてのデータの処理が試行されます。

_最初に報告されたのは 2024年8月14日（PT）です。_
