# Workfront Fusion：単一のモジュールから作成された重複プロジェクト

<!--Fusion, WF TOCs-->

シナリオの実行中にプロジェクトの作成モジュールを使用すると、2 つの同じプロジェクトが作成されます。 これは、Workfrontでプロジェクトを使用する際に混乱を引き起こす可能性があります。

_最初に報告されたのは 2023年10月11日（PT）です。_
