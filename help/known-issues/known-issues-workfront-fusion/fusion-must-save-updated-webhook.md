# Workfront Fusion：Webhook のアップデートが自動保存されない

Workfront／Watch イベントモジュールなどでユーザーが Webhook に変更を加えても、その変更内容は自動的には保存されません。

Webhook への変更を保持するには、ユーザーは、**シナリオを保存する必要があります**。

**回避策**

シナリオを保存します。

_最初に報告されたのは、2025年7月17日（PT）です。_
