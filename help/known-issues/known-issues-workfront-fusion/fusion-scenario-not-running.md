# Workfront Fusion：シナリオが実行されない

シナリオが実行されなくなった場合、夏時間の問題が原因である可能性があります。

**回避策**

影響を受けるシナリオを非アクティブ化して再アクティブ化します。

_最初に報告されたのは 2025年3月18日（PT）です。_
