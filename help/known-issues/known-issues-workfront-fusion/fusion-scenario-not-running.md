# Workfront Fusion：シナリオが実行されていない

>[!NOTE]
>
>この問題は解決されました。

シナリオが実行されなくなった場合は、夏時間の問題が原因の可能性があります。

**回避策**

影響を受けるシナリオを非アクティブ化してから再アクティブ化します。

_最初に報告されたのは 2025年3月18日（PT）です。_
