# 統合：Workfront for G Suite を開けない」

ユーザーが G Suite アプリを使用していて、Workfront for G Suite アドオンを開こうとすると、アドオンが開かず、「エラーが発生しました。もう一度試してください。」というメッセージが表示されます。もう一度やりなおしても、効果ありません。

_最初に報告されたのは 2022年12月1日です。_
