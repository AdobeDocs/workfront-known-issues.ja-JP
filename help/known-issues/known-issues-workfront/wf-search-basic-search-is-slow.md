# 検索：検索が非常に遅い

ユーザーが基本検索（検索アイコン）を使用して検索する場合、検索結果の返される時間が非常に長くなります。

**回避策**

詳細検索を使用してください。「詳細検索」へのリンクは、検索アイコンをクリックすると表示されるドロップダウンの下部にあります。

_最初に報告されたのは 2022年11月1日です。_
