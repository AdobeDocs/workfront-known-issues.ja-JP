# 検索： [!UICONTROL 検索] 非常に遅い

>[!NOTE]
>
>この問題は 2022 年 11 月 3 日に修正されました。

ユーザーが基本検索 ( [!UICONTROL 検索] アイコン )、検索結果が非常に遅く返される場合。

**回避策**

[!UICONTROL 詳細検索]を使用してください。リンク先 [!UICONTROL 詳細検索] は、 [!UICONTROL 検索] アイコン

_最初に報告されたのは 2022年11月1日です。_
