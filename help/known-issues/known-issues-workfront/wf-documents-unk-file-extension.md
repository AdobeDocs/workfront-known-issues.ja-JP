# ドキュメント：統合からドキュメントをリンクする際の UNK ファイル拡張子

<!--WF and WFP-->

>[!NOTE]
>
>この問題は、Workfront の問題ではないため、クローズされました。

ユーザーがドキュメント統合からドキュメントをリンクすると、ドキュメントは UNK ファイル拡張子でWorkfrontに表示されます。

_最初に報告されたのは 2024年5月12日（PT）です。_
