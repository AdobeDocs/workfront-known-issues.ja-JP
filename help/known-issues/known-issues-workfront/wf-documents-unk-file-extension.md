# ドキュメント：統合からドキュメントをリンクする際に UNK ファイル拡張子付きで表示される

<!--WF and WFP-->

>[!NOTE]
>
>この問題は、Workfront の問題ではないため、クローズされました。

ユーザーがドキュメント統合からドキュメントをリンクすると、そのドキュメントは Workfront に UNK ファイル拡張子付きで表示されます。

_最初に報告されたのは、2024年5月12日（PT）です。_
