# タスク：先行タスク情報にリンクが含まれない

>[!NOTE]
>
>このイシューは 2024年4月23日（PT）に修正されました。

ユーザーが先行タスクにカーソルを合わせる際、先行タスクに関する詳細情報が表示されますが、その情報には先行タスクへのリンクは含まれません。

_最初に報告されたのは 2024年3月22日（PT）です。_
