# リクエスト：リクエストのコピー時に「システム管理者である必要があります」というエラーが発生する

>[!NOTE]
>
>この問題は2026年2月3日（PT）に修正されました。

ユーザーがリクエストをコピーして保存しようとすると、リクエストは保存されず、次のメッセージが表示されます。

「このカスタムデータパラメーター値を変更するには、システム管理者である必要があります。」

_最初に報告されたのは、2025年1月12日（PT）です。_
