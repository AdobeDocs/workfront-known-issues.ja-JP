# ユーザー：ユーザーの一括更新が非常に遅い

>[!NOTE]
>
>この問題は、サンドボックス環境に関してのみ報告されています。

ユーザーがユーザーを一括編集しようとすると、編集ウィンドウが開くまでに非常に長い時間がかかります。 これは 2 人のユーザーにも当てはまりますが、より多くのユーザーが時間がかかります。

_最初に報告されたのは 2024年12月6日（PT）です。_
