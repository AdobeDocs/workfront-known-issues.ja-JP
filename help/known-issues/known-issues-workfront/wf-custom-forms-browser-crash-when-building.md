# カスタムフォーム：大きなフォームを作成する際にブラウザーがクラッシュする

ユーザーがフォームデザイナーで大きなフォームを作成または編集すると、ブラウザーがクラッシュする可能性があり、ユーザーはフォームを作成または編集できません。

これは、フォームデザイナーに影響を与えています。 フォームは、オブジェクトで期待どおりに使用できます。

_最初に報告されたのは 2024年7月29日（PT）です。_
