# リソースプランニング：「[!UICONTROL 詳細]」ボタンが表示されない

ユーザーがリソースプランナーを表示中に、「[!UICONTROL 詳細]」ボタンをクリックしようとしても、ボタンは表示されません。

**回避策**

「[!UICONTROL 詳細]」ボタンが表示されるエリアをクリックします。ボタンは表示されませんが、クリックできます。

_最初に報告されたのは 2024年6月5日（PT）です。_
