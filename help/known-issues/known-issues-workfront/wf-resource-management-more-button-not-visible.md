# リソース計画： [!UICONTROL 詳細] ボタンが表示されない

ユーザーがリソースプランナーを表示中に [!UICONTROL 詳細] ボタンが表示されます。ボタンは表示されません。

**回避策**

次の領域をクリックします [!UICONTROL 詳細] ボタンが必要です。 ボタンは表示されませんが、クリックできます。

_最初に報告されたのは 2024年6月5日（PT）です。_
