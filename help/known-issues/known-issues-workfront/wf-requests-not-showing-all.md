# リクエスト：「すべて」ビューにすべてのリクエストが表示されない

>[!NOTE]
>
>* この問題は、新しい要求元エクスペリエンスで報告されています。


ユーザーが新しいリクエストエクスペリエンスでリクエストリストを表示しており、ビューがすべてのリクエストを表示するように設定されている場合、一部のリクエストのみが表示されます。

_最初に報告されたのは、2025年12月16日（PT）です。_
