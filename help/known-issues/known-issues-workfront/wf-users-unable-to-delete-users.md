# ユーザー：ユーザーを削除できない

>[!NOTE]
>
>この問題は、2024年5月16日（PT）に修正されました。

ユーザーがユーザーを削除しようとしても、そのユーザーは削除されません。 この問題は、Adobe Admin Console に移行した組織で報告されています。

_最初に報告されたのは 2024年3月11日（PT）です。_
