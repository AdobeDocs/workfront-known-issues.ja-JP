# ユーザー：ユーザーを削除できない

ユーザーがユーザーを削除しようとしても、そのユーザーは削除されません。 この問題は、Adobe Admin Consoleに移行した組織で報告されています。

_最初に報告されたのは 2024年3月11日（PT）です。_
