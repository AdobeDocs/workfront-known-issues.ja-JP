# プルーフ：PDF からプルーフを作成できない

<!--WF and WFP TOCs-->

>[!NOTE]
>
>この問題は 2023年8月24日（PT）に修正されました。

ユーザーが PDF からプルーフを作成しようとすると、プルーフは生成されず、プルーフの生成に失敗したというメッセージがユーザーに表示されます。

_最初に報告されたのは 2023年8月3日（PT）です。_
