# 配達確認：配達確認をPDFから作成できません

<!--WF and WFP TOCs-->

ユーザーがPDFから配達確認を作成しようとすると、配達確認は生成されず、配達確認の生成に失敗したというメッセージがユーザーに表示されます。

_最初に報告されたのは 2023年8月3日（PT）です。_
