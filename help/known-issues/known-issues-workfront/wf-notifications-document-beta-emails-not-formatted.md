# 通知：ドキュメント承認ベータ版からの通知の形式が正しくない

ユーザーが新しいドキュメント承認（ベータ版）からメールを受信すると、そのメールの形式が正しくありません。

この問題は、Windows 版 Microsoft Outlook で報告されています。

_最初に報告されたのは 2024年3月14日（PT）です。_
