# カスタムフォーム：計算フィールドに管理者以外のユーザーのデータが表示されない

>[!NOTE]
>
>この問題は、2026年2月19日（PT）に修正されました。

Workfront管理者以外のユーザーが、フォームの管理者専用セクションのデータから取り込むフィールドを持つカスタムフォームを表示した場合、そのフィールドには計算された値が表示されません。 ユーザーが式の再計算を実行すると、値は表示されますが、更新時には失われます。

Workfront管理者は、フィールド値を期待どおりに表示できます。

_最初に報告されたのは、2026年1月13日（PT）です。_
