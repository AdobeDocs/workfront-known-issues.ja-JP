# カスタムフォーム：管理者以外のデータを表示しない計算フィールド

>[!NOTE]
>
>この問題は2026年2月19日（PT）に修正されました。

Workfront管理者ではないユーザーが、フォームの管理者のみのセクションのデータから取得したフィールドを含むカスタムフォームを表示すると、そのフィールドには計算値が表示されません。 ユーザーが「エクスプレッションを再計算」を実行すると、値は表示されますが、更新時に失われます。

Workfront管理者は、フィールド値を期待どおりに表示できます。

_最初に報告されたのは、2026年1月13日（PT）です。_
