# ボード：ステータスを変更した際に、カードが適切な列に移動しない

接続されているカードのリンクされたオブジェクトのステータスをオブジェクト上で直接変更すると、カードは適切な列に移動しません。カード上のオブジェクトのステータスを変更した場合や、カードを新しい列にドラッグした場合、カードは期待どおりに動作します。

_最初に報告されたのは 2023年11月28日（PT）です。_
