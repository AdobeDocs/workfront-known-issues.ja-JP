# プルーフ：GIF が静的プルーフとしてレンダリングされる

>[!NOTE]
>
>この問題は、2024年1月25日（PT）に修正されました。

ユーザーが GIF からプルーフを作成しようとすると、静止画像として GIF の最初のフレームのみを使用してプルーフが作成されます。

_最初に報告されたのは 2023年1月22日（PT）です。_
