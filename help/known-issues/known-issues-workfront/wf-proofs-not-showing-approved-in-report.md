# 配達確認：配達確認が承認済みとしてレポートに表示されない

ユーザーが配達確認の承認の決定を含むレポートを表示している場合、レポートには [!UICONTROL 承認済み] 承認されたすべての配達確認に関する決定 ユーザーが配達確認自体を表示すると、その配達確認が承認されたことを確認できます。

_最初に報告されたのは 2023年6月26日（PT）です。_
