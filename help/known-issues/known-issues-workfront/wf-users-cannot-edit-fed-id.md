# ユーザー：ユーザーの連合 ID を編集できない

>[!NOTE]
>
>この問題は、アドビの ID 管理システムへの移行がまだ完了していない組織に影響します。

管理者がユーザーの連合 ID を更新しようとすると、この情報のフィールドを使用できないため、管理者は更新できません。

つまり、ユーザーは Workfront にログインできません。

_最初に報告されたのは 2025年9月9日（PT）です。_
