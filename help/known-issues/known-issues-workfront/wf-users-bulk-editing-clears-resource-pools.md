# ユーザー：ユーザーを一括編集すると、リソースプールと直属の部下が削除される

>[!NOTE]
>
>この問題は、2025年12月9日（PT）に修正されました。

ユーザーがユーザーの一括編集時に、「リソースプール」フィールドをクリックしたものの値を入力しない場合、編集されたすべてのユーザーのリソースプールが削除されます。

一括編集すると、編集したユーザーから直属の部下が削除される場合もあります。

_最初に報告されたのは、2025年11月21日（PT）です。_
