# ワークロードバランサー：ユーザーあたりの時間が正しく計算されない

>[!NOTE]
>
>この問題は、2026年2月6日（PT）に修正されました。

ユーザーがワークロードバランサーでユーザーの時間を表示すると、その時間が正しく計算されない場合があります。 例えば、週単位のビューには 1 つの時間数が表示されますが、4 週単位のビューには、その週の異なる時間数が表示されます。

_最初に報告されたのは、2026年1月27日（PT）です。_
