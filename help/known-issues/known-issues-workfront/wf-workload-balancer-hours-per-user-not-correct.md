# ワークロードバランサー：ユーザーあたりの時間が正しく計算されない

>[!NOTE]
>
>この問題は2026年2月6日（PT）に修正されました。

ユーザーがワークロードバランサーでユーザーの時間を表示している場合、それらの時間は誤って計算される可能性があります。 例えば、週ビューでは1つの時間の計算が表示されますが、4週間ビューでは、その週の時間数が異なります。

_最初に報告されたのは2026年1月27日です。_
