# 統合：SSO 使用時にユーザーが間違ったリクエストエラーが表示される

>[!NOTE]
>
>この問題は、2024年5月24日（PT）に修正されました。

ユーザーが SSO を有効にして Outlook 用 Workfront プラグインにログインしようとすると、「間違ったリクエスト」ページが表示され、ログインできません。

_最初に報告されたのは 2024年5月14日（PT）です。_
