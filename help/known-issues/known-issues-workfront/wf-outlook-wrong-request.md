# 統合：SSO を使用する際に、ユーザーに間違ったリクエストエラーが発生する

SSO が有効な Outlook 用Workfront プラグインにログインしようとすると、「Wrong Request」ページが表示され、ログインできません。

_最初に報告されたのは 2024年5月14日（PT）です。_
