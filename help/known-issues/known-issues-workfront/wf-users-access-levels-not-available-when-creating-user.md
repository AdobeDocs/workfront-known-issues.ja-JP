# ユーザー：ユーザーの作成時に、低いアクセスレベルを使用できない

>[!NOTE]
>
>このイシューは 2024年3月14日（PT）に修正されました。

ユーザーが別のユーザーを作成する際、新しく作成したユーザーが利用できるのは、最初のユーザーのアクセスレベルのみです。作成するユーザーよりも権限の低いすべてのアクセスレベルを、新しいユーザーに割り当てる必要があります。

_最初に報告されたのは 2024年1月30日（PT）です。_
