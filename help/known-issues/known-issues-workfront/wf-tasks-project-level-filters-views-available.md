# タスク：タスクに使用できるプロジェクトレベルのフィルターとビュー

>[!NOTE]
>
>この問題は、2025年2月21日（PT）に修正されました。

タスクのフィルターリストまたはビューリストを開くと、プロジェクトレベルで作成されたフィルターとビューがオプションのリストに表示されます。

_最初に報告されたのは 2025年1月27日（PT）です。_
