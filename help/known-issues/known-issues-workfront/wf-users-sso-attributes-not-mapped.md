# ユーザー：ユーザー属性が SSO からマッピングされない

>[!NOTE]
>
>この問題は、2024年5月23日（PT）に修正されました。

SSO からの属性は、設定が正しく行われている場合でも、Workfront に表示されません。

この問題は、Adobe Admin Console にオンボードした組織で報告されています。

_最初に報告されたのは 2024年4月15日（PT）です。_
