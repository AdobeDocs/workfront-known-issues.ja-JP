# ユーザー：ユーザー属性は SSO からマッピングされません

設定で正しく設定されていても、SSO の属性がWorkfrontに表示されません。

この問題は、Adobe Admin Consoleにオンボーディングされた組織に関して報告されています。

_最初に報告されたのは 2024年4月15日（PT）です。_
