# API:リクエストがタイムアウトする

Workfront API を介して行われたリクエストがタイムアウトする。

FusionはWorkfront APIを使用するため、Fusionシナリオはこれらのタイムアウトの影響を受ける可能性があります。

_最初に報告されたのは 2024年6月21日（PT）です。_
