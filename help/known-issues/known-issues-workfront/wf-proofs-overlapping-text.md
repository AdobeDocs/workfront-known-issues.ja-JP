# プルーフ：プルーフで重なっているテキスト

プルーフビューアーでプルーフを表示すると、元のドキュメントでテキストが重複していない場合でも、テキストが重なっていることに気が付きます。

この問題は、PPTX（PowerPoint）ファイルから作成されたプルーフに関して報告されています。

_最初に報告されたのは 2023年7月18日（PT）です。_
