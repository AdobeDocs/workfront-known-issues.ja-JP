# アクセスレベル：設定領域のアクセスレベルを複製

ユーザーが「設定」領域でアクセスレベルのリストを表示すると、一部のアクセスレベルが重複して表示されます。 これは、ビルトインレベルにのみ適用されます。

この問題は、組織のライセンスに変更を加えた後に報告されています。

_最初に報告されたのは 2024年7月18日（PT）です。_
