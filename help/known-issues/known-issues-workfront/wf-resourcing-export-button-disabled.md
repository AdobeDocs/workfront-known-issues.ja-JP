# リソースプランナー：「書き出し」ボタンが無効化

>[!NOTE]
>
>この問題は 2025年6月9日（PT）に修正されました。

ユーザーがリソースプランナーから書き出しを試みると、ボタンが無効（グレー表示）になり、ユーザーは書き出すことができません。

_最初に報告されたのは 2025年5月20日（PT）です。_
