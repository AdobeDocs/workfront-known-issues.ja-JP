# ユーザー：システム管理者の変更時に重複アカウントが作成される

システムのプライマリ管理者が別のユーザーに変更されると、そのユーザーのアカウントの複製が「Admin User」という名前で、同じメールアドレスで作成されます。

_最初に報告されたのは 2024年12月6日（PT）です。_
