# タスク：タスクに対するインライン編集が保存されない

>[!NOTE]
>
>* この問題は、2025年12月15日（PT）に修正されました。
>* この問題は、レガシータスクエクスペリエンスで発生します。

ユーザーがレガシータスクエクスペリエンスでタスクにインライン変更を加えて保存すると、変更内容が保存されません。

**回避策**

一部の変更では、新しいタスクエクスペリエンスを使用できます。

_最初に報告されたのは、2025年12月4日（PT）です。_
