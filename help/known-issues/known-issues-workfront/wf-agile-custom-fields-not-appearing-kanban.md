# アジャイル：かんばんカードにカスタムフィールドが表示されない

>[!NOTE]
>
>このイシューは 2024年4月18日（PT）に修正されました。

ユーザーがカスタムフィールドを含めるように設定されたかんばんボードを表示している際、これらのカスタムフィールドは表示されない場合があります。

_最初に報告されたのは 2024年3月6日（PT）です。_
