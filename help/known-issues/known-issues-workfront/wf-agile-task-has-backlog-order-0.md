# アジャイル：タスクのかんばんバックログオーダーは 0 です

ユーザーがかんばんチームのバックログを表示している場合、1 つ以上のタスクに 0 のバックログ順序が表示されます。

_最初に報告されたのは 2022年10月7日です。_
