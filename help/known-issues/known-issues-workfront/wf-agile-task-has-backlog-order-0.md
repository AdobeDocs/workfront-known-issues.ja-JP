# Agile：タスクのかんばんバックログの順序が 0 になる

>[!NOTE]
>
>この問題は、2022年12月1日に修正されました。

ユーザーがかんばんチームのバックログを表示していると、1 つ以上のタスクでバックログの順序が 0 と表示されます。

_最初に報告されたのは 2022年10月7日です。_
