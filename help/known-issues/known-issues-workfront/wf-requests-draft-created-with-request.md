# リクエスト：ドラフトをリクエストと共に作成すると問題が発生する

新しい要求エクスペリエンスでユーザーが新しい要求を作成して送信すると、ドラフトも作成されます。 ドラフトには送信されたリクエストの名前の最初の文字のみが表示され、リクエストを開こうとすると、「取得に失敗しました」エラーが発生します。

また、送信されたリクエストが削除されると、削除されたように見えますが、Workfrontの「リクエスト」領域には表示されたままになります。 ユーザーが要求を再度開こうとすると、要求が存在しないことを示すメッセージが表示されます。

_最初に報告されたのは 2025年1月6日（PT）です。_
