# リクエスト：リクエストと一緒に作成されたドラフトが問題を引き起こす

>[!NOTE]
>
>この問題は2026年2月10日（PT）に修正されました。

ユーザーが新しいリクエストエクスペリエンスで新しいリクエストを作成して送信すると、ドラフトも作成されます。 ドラフトには、送信されたリクエストの名前の最初の文字のみが表示され、それを開こうとすると、「取得に失敗しました」エラーが発生します。

また、送信されたリクエストが削除された場合、そのリクエストは削除されたように見えますが、Workfrontのリクエスト領域には表示されたままになります。 ユーザーがリクエストを再度開こうとすると、リクエストが存在しないことを示すメッセージが表示されます。

_最初に報告されたのは、2025年1月6日（PT）です。_
