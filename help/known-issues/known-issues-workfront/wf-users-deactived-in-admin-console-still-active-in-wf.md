# ユーザー：Adobe Admin Consoleから削除されたユーザーは、引き続きWorkfrontでアクティブです

管理者がAdobe Admin Consoleでユーザーを削除しても、そのユーザーはWorkfrontから削除されません。 このユーザーは、Workfrontで引き続き表示され、アクティブです。

_最初に報告されたのは2026年2月3日です。_
