# ユーザー：Adobe Admin Consoleから削除されたユーザーは、引き続きWorkfrontでアクティブです

管理者がAdobe Admin Consoleでユーザーを削除しても、そのユーザーはWorkfrontから削除されません。 ユーザーはWorkfrontに表示され、アクティブなままです。

_最初に報告されたのは、2026年2月3日（PT）です。_
