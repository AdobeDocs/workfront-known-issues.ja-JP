# レポート：グループ名に / または \ があると「表示するデータがない」

>[!NOTE]
>
>この問題は 2024年7月4日（PT）にクローズされました。

ユーザーがレポート内のグラフを表示し、グラフ上のグループ化をクリックし、そのグループ名に / または \ が含まれていると、開いた詳細にはグループ化の項目が表示されず、「表示するデータがありません」というメッセージが表示されます。

_最初に報告されたのは 2024年6月17日（PT）です。_
