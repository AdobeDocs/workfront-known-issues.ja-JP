# 割り当て：タスクの割り当てに失敗し、継承された権限が削除される

タスクをチームに割り当てると、割り当てが失敗し、エラーが返されます。 さらに、継承された権限は削除され、割り当てようとしたタスクに対するユーザーのアクセス権が失われます。

_最初に報告されたのは 2024年9月5日（PT）です。_
