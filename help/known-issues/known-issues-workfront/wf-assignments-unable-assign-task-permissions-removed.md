# 割り当て：タスクの割り当てが失敗し、継承した権限が削除される

チームにタスクを割り当てると、割り当てが失敗し、エラーが返されます。さらに、継承した権限が削除され、ユーザーが割り当てようとしていたタスクへのアクセス権が失われます。

_最初に報告されたのは 2024年9月5日（PT）です。_
