# ナビゲーション：「戻る」ボタンが前のページに戻らない

Workfrontでユーザーがブラウザーの「戻る」ボタンをクリックすると、次のいずれかが発生する場合があります。

* ブラウザータブの名前は変更されますが、ページは変更されません。 戻るボタンをもう一度クリックすると、問題が解決する場合があります。
* ユーザーは、ブラウザーのランディングページに移動します。

_最初に報告されたのは 2024年6月3日（PT）です。_
