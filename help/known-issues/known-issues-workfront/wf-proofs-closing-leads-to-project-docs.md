# プルーフ：プルーフを閉じると、製品ドキュメントページに移動する

>[!NOTE]
>
>この問題は 2024年7月18日（PT）に修正されました。

ユーザーがプルーフを表示してそのプルーフを閉じると、プルーフを開いたページではなく、プロジェクトドキュメントページに移動します。

_最初に報告されたのは 2024年6月13日（PT）です。_
