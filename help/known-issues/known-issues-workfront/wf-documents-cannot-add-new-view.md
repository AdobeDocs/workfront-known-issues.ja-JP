# ドキュメント：ドキュメントリストのビューを作成できない

>[!NOTE]
>
>この問題は 2024年8月1日（PT）に修正されました。

ユーザーがドキュメントリストで新しいビューを作成しようとすると、画面が空白になり、ユーザーはビューを作成できません。

既存のビューは期待どおりに動作します。

**回避策**

管理者は、設定／インターフェイス／ビューエリアで、新規ドキュメントビューを追加できます。

_最初に報告されたのは 2024年7月9日（PT）です。_
