# 通知：Workfront モバイルアプリへのプッシュ通知が機能しない

Workfront モバイルアプリへのプッシュ通知をトリガーするイベントが発生しても、通知が受信されません。

この問題は、通知がメールまたはアプリ内通知で受信された場合でも発生する場合があります。

_最初に報告されたのは 2024年12月17日（PT）です。_
