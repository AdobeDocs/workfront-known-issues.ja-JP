# ドキュメント：ドキュメントの更新でプルーフコメントが表示されない

ユーザーがプルーフにコメントを追加しても、そのコメントはWorkfrontのプルーフのドキュメント更新領域にコピーされません。

_最初に報告されたのは 2024年7月23日（PT）です。_
