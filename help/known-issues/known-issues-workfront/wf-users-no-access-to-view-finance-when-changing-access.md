# ユーザー：ユーザーアクセスを更新できない

ユーザーがプロジェクトでの「表示」から「管理」に別のユーザーのアクセス権を更新しようとすると、ユーザーはアクセス権を更新できず、次のメッセージが表示されます。

「このプロジェクトの財務を表示するために十分なアクセス権限がありません。」

_最初に報告されたのは 2025年12月11日（PT）です。_
