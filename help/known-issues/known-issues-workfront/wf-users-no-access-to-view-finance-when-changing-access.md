# ユーザー：ユーザーアクセスを更新できない

>[!NOTE]
>
>この問題は、設計どおりの動作であるため、クローズされました。

ユーザーがプロジェクトで別のユーザーのアクセス権を「表示」から「管理」に更新しようとすると、アクセス権を更新できず、次のメッセージが表示されます。

「このプロジェクトの財務を表示するために十分なアクセス権限がありません。」

_最初に報告されたのは、2025年12月11日（PT）です。_
