# プルーフ：プルーフ名をクリックすると、空白の画面に移動する

Web プルーフビューアでユーザーがプルーフ名をクリックしてプルーフを開くと、プルーフではなく空白の画面に移動します。

_最初に報告されたのは 2024年6月20日（PT）です。_
