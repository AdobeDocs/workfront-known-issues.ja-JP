# 配達確認：リストまたは箇条書きのテキストを選択できません

<!--WF and WFP TOCs-->

ユーザーが校正ビューアで配達確認を表示していて、リスト内または箇条書き内のテキストを選択しようとすると、テキスト選択ツールは無効になり、テキストは選択できません。

_最初に報告されたのは 2023年8月1日（PT）です。_
