# プロジェクト：問題から変換されたプロジェクトは、ポートフォリオまたはプログラムから権限を継承しません

問題をプロジェクトに変換すると、新しいプロジェクトは、そのプロジェクトが作成されるポートフォリオまたはプログラムから権限を継承しません。

_最初に報告されたのは 2022年10月12日です。_
