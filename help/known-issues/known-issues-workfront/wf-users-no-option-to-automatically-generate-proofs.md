# ユーザー：ユーザープロファイルにプルーフを自動生成するオプションがない

>[!NOTE]
>
>この問題は、2025年12月17日（PT）に修正されました。

ユーザーがユーザープロファイルを表示している場合、「ドキュメントのアップロード時に自動的にプルーフを生成」するオプションは使用できません。 この問題は、オプションのすべての前提条件が満たされている場合でも発生する可能性があります。

_最初に報告されたのは、2025年12月5日（PT）です。_
