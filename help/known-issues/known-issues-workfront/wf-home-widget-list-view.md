# ホーム：「widget.list.view」は、ユーザーがホームウィジェットで列を移動すると作成されます

管理者が「設定」領域でビューのリストを表示すると、「widget.list.view」と呼ばれる多くのビューが表示されます。 これらのビューは、ユーザーがホームウィジェットの列の配置を変更したときに作成されます。

_最初に報告されたのは 2024年8月12日（PT）です。_
