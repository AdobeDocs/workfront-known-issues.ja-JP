# 更新：コメント内のリンクが空白の画面に移動する

>[!NOTE]
>
>このイシューは 2024年4月25日（PT）に修正されました。

ユーザーがコメント内のリンクをクリックすると、空白の画面に移動します。

**回避策**

リンクをコピーして、別のページの URL バーに貼り付けます。

_最初に報告されたのは 2024年4月18日（PT）です。_
