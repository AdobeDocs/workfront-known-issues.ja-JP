# [!UICONTROL ホーム]:で問題の割り当てを削除できません [!UICONTROL ホーム]

ユーザーが [!UICONTROL ホームの作業リスト] また、割り当てを削除しようとすると、ページが更新され、割り当ては削除されませんでした。

**回避策**

イシューを開き、イシュー自体から割り当てを削除します。

_最初に報告されたのは 2022年6月1日です。_
