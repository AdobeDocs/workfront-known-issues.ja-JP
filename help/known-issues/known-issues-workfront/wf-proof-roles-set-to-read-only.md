# プルーフ：プルーフの役割が[!UICONTROL 読み取り専用]に変更される

>[!NOTE]
>
>この問題は 2023年3月16日に修正されました。

プルーフの役割が[!UICONTROL 読み取り専用]以外のつユーザーは、自動的に[!UICONTROL 読み取り専用]に変更されます。この変更を示す通知やログはありません。

_最初に報告されたのは 2023年2月8日（PT）です。_
