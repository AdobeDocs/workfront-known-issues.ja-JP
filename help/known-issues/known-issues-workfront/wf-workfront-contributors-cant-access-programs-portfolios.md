# Workfront：投稿者ユーザーがプログラムやPortfolioにアクセスできない

新しいコントリビューターアクセスレベルを持つユーザーは、両方へのアクセス権がアクセスレベルに含まれているにもかかわらず、プログラムやPortfolioにアクセスできません。

_最初に報告されたのは 2024年10月16日（PT）です。_
