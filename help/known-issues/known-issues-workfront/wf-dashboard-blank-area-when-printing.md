# ダッシュボード：ダッシュボードをPDFに印刷する際の空白の大きい余白

ユーザーが横向きでPDFに対してPDFを印刷すると、ダッシュボードの右側に大きな空白の余白が表示されます。

**回避策**

縦長モードで印刷すると、正しい余白が表示されます。

_最初に報告されたのは 2023年10月23日（PT）です。_
