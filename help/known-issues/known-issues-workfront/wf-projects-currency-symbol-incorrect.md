# プロジェクト：書き出し時の通貨記号が正しくない

ユーザーが問題を書き出す場合、書き出しの通貨記号がプロジェクトまたは問題に設定された通貨と一致しません。

_最初に報告されたのは 2024年7月12日（PT）です。_
