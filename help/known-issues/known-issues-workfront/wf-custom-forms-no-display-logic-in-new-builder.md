# カスタムフォーム：新しいフォームデザイナー（ベータ版）でカスタムフォームが編集されると、表示ロジックが失われます

現在、新しいフォームデザイナー（ベータ版）では表示ロジックを使用できません。 したがって、新しいフォームデザイナー（ベータ版）で既存のフォームを編集すると、フォーム上の既存の表示ロジックは失われます。

_最初に報告されたのは 2023年5月23日（PT）です。_
