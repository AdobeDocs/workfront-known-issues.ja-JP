# タスク：タスクの編集ダイアログで行った編集が保存されない

>[!NOTE]
>
>* この問題は、レガシータスクエクスペリエンスで発生します。
>* 製品チームは現在、この問題の解決策を評価中です。製品の強化が必要になる可能性があります。 製品の機能強化は、メンテナンス更新ではなく、製品のお知らせで連絡します。

ユーザーがレガシータスクダイアログでタスクの「編集」セクションに変更を加えて保存すると、変更内容が保存されません。

**回避策**

一部の変更では、新しいタスクエクスペリエンスを使用できます。

_最初に報告されたのは、2025年11月24日（PT）です。_
