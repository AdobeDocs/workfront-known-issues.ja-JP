# タスク：移動すると、前のプロジェクトに記録された時間が残る

タスクを新しいプロジェクトに移動すると、元のプロジェクトのプロジェクト時間の合計には、移動したタスクの時間が含まれます。新しいプロジェクトでも、タスクの記録された時間は適切に保持されます。

_最初に報告されたのは 2024年12月11日（PT）です。_
