# タイムシート：コントリビューターのユーザーに「承認用に送信」ボタンがある

タイムシートに時間を入力できないにもかかわらず、コントリビューターのアクセスレベルを持つユーザーには、クリックできる「承認用に送信」ボタンが表示されます。

_最初に報告されたのは 2024年9月9日（PT）です。_
