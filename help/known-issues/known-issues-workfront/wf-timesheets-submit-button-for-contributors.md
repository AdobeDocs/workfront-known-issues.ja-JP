# タイムシート：投稿者ユーザーには「承認用に送信」ボタンがあります

タイムシートに時間を入力することはできませんが、投稿者アクセスレベルを持つユーザーは、クリックできる「承認用に送信」ボタンを使用できます。

_最初に報告されたのは 2024年9月9日（PT）です。_
