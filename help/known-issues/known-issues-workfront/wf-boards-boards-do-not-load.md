# [!UICONTROL ボード]: [!UICONTROL ボード] ページが読み込まれません

ユーザーが [!UICONTROL ボード]、のヘッダー [!UICONTROL ボード] が短く表示された後、が消え、ユーザーには空白のページのみが表示されます。
