# 配達確認：配達確認をクリックしたときの「画像なし」メッセージ

デスクトップ校正ビューアのユーザーがコメントをクリックすると、コメントに関連付けられた画像が開かず、次のメッセージが表示されます。

&quot;[!UICONTROL このコメントに関連付けられた画像はありません。]&quot;
