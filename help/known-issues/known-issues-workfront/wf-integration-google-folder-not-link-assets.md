# 統合：からのフォルダーのリンク [!DNL Google Drive] はフォルダー内のアセットをリンクしません

ユーザーがでフォルダーにリンクした場合 [!DNL Google Drive] Workfrontに対しては、そのフォルダー内のアセットはリンクされません。 フォルダー内のアセットの数は 0 と表示されます。

_最初に報告されたのは 2024年6月1日（PT）です。_
