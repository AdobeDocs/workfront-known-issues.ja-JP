# 統合：[!DNL Google Drive] からフォルダーをリンクしても、フォルダー内のアセットがリンクされない

ユーザーが [!DNL Google Drive] 内のフォルダーを Workfront にリンクしても、そのフォルダー内のアセットはリンクされません。フォルダー内のアセットの数は 0 として表示されます。

**回避策**

Google接続を再認証します。

_最初に報告されたのは 2024年6月1日（PT）です。_
