# カスタムフォーム：Formsのデフォルトはプロジェクトフォームです

ユーザーがカスタムフォームを作成する際、フォームのオブジェクトタイプを選択すると、オブジェクトタイプは無視され、フォームは Project のカスタムフォームとして作成されます。

_最初に報告されたのは 2024年8月12日（PT）です。_
