# リクエスト：リクエストの入力時にドキュメントをアップロードできない

>[!NOTE]
>
>この問題は2023年2月23日（PT）に修正されました。

ユーザーがリクエストを入力し、ドキュメントをアップロードしようとすると、ドキュメントはアップロードされず、次のエラーが表示されます。

「アップロードエラー：作成後に小さいファイルを試すか、ファイルを添付してください。」

_最初に報告されたのは2026年2月6日です。_
