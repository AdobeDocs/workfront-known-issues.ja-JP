# リクエスト：リクエストの入力時にドキュメントをアップロードできない

>[!NOTE]
>
>この問題は、2023年2月23日（PT）に修正されました。

ユーザーがリクエストに入力中にドキュメントをアップロードしようとすると、ドキュメントがアップロードされず、次のエラーが表示されます。

「アップロードエラー：作成後、小さいファイルを試すか、ファイルを添付してください。」

_最初に報告されたのは、2026年2月6日（PT）です。_
