# タイムシート：タイムシートで時間を追加する際に「[!UICONTROL 請求済み時間は既に存在します]」エラーが発生する

>[!NOTE]
>
>この問題は 2022年8月25日に修正されました。

ユーザーがタイムシートでタスクに時間を追加しようとすると、時間が追加されず、次のエラーが表示されます。

[!UICONTROL 請求済み時間は、既に（日付）のこのプロジェクトに存在します。]

_最初に報告されたのは 2022年7月22日です。_
