# Workfront計画：リクエストが承認されると、リクエストのステータスが「進行中」と表示される

ユーザーがリクエストリストで完了したリクエストを表示すると、表示されるリクエストステータスは「完了」ではなく「進行中」になります。

ユーザーがリクエストを開くと、リクエストのステータスが「完了」であることがわかります。 この問題は、表示されるステータスにのみ影響し、リクエストがどのステータスにあるかは影響しません。

_最初に報告されたのは2026年1月27日です。_
