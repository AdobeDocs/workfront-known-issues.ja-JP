# ボード：エラー [!UICONTROL コメント] カードのセクション

ユーザーがカードを表示し、 [!UICONTROL コメント]、セクション、コメントは表示されず、ユーザーに次のエラーが表示されます。

「[!UICONTROL 異常が発生しました。後でもう一度試してください。]」
