# ボード：カードの「[!UICONTROL コメント]」セクションでエラーが発生する

>[!NOTE]
>
>この問題は、2024年1月18日（PT）に修正されました。

ユーザーがカードを表示し、「[!UICONTROL コメント]」セクションまでスクロールすると、コメントは表示されず、次のエラーが表示されます。

「[!UICONTROL 異常が発生しました。後でもう一度試してください。]」

_最初に報告されたのは 2023年11月23日（PT）です。_
