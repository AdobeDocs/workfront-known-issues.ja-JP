# レポート：使用率レポートの予算時間は、API で報告された予算時間と一致しません

ユーザーが特定のプロジェクトの RPBGHR オブジェクトに対して API 呼び出しを行い、その呼び出しの結果をそのプロジェクトの使用率レポートと比較すると、結果は一致しません。

_最初に報告されたのは 2023年9月20日（PT）です。_
