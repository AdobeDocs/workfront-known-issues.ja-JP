# レポート：稼働率レポートの予算計上時間数が API を通じてレポートされた予算計上時間数と一致しない

>[!NOTE]
>
>この問題は、2023年12月21日に修正されました。

ユーザーが特定のプロジェクトの RPBGHR オブジェクトに対して API 呼び出しを行い、その呼び出しの結果をそのプロジェクトの稼働率レポートと比較しても、結果が一致しません。

_最初に報告されたのは 2023年9月20日（PT）です。_
