# タスク：承認エリアが左側のナビゲーションから消える

ユーザーがタスクを表示すると、左側のナビゲーションから承認エリアが表示されなくなる場合があります。承認エリアを削除してレイアウトテンプレートに追加すると、問題は解決しますが、問題が再発する可能性があります。

_最初に報告されたのは 2025年6月18日（PT）です。_
