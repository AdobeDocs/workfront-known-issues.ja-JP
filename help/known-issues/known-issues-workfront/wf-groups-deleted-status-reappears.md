# グループ：削除したグループのステータスが再表示される

グループのステータスを削除すると、グループの更新にステータスと削除時間が表示されなくなりますが、最終的にはユーザーによる変更なしで再表示されます。

_最初に報告されたのは 2024年6月26日（PT）です。_
