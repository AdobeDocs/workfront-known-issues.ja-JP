# タスク／イシュー：割り当てられたユーザーがオブジェクトへのアクセス権を受け取らない

>[!NOTE]
>
>この問題はクローズされました。

ユーザーが作業オブジェクトに割り当てる際、そのユーザーにオブジェクトへのアクセス権が届かず、割り当てを完了できない場合があります。ユーザーはアクセス権をリクエストできますが、これにより作業アイテムの完了に時間がかかり、複雑さが増します。

**回避策**

次のいずれかの操作を行います。

* ユーザーの割り当て解除と再割り当て
* ユーザーへの手動でのアクセス権の付与

_最初に報告されたのは 2023年7月です。_
