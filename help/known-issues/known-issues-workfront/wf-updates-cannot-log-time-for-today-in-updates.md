# [!UICONTROL 更新]:現在の日の時刻を [!UICONTROL 更新] 領域

ユーザーが、 [!UICONTROL 更新] 「 」領域では、現在の日は灰色表示になっており、ユーザーは時間を入力できません。

**回避策**

ブラウザーのキャッシュをクリアするか、匿名ウィンドウを使用すると、問題を一時的に解決できます。

_最初に報告されたのは 2023年3月7日です。_
