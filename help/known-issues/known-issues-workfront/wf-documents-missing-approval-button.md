# ドキュメント：ドキュメントに「承認オプション」ボタンが表示されない

ドキュメントに対して承認者として割り当てられたユーザーがドキュメントに移動して承認決定を行うと、「承認オプション」ボタンが表示されません。

_最初に報告されたのは 2024年10月24日（PT）です。_
