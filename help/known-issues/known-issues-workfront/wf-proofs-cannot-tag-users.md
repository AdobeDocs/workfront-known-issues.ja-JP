# プルーフ：特定のユーザーにタグ付けできない

ユーザーがプルーフで別のユーザーをタグ付けしようとすると、typeahead フィールドで一部のユーザーを使用できない場合があります。

_最初に報告されたのは 2025年1月10日（PT）です。_
