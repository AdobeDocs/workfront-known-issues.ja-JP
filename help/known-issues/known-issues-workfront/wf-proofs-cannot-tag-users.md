# プルーフ：特定のユーザーにタグ付けできない

>[!NOTE]
>
>この問題は、2025年2月7日（PT）に修正されました。

ユーザーがプルーフで別のユーザーをタグ付けしようとすると、一部のユーザーが先行入力フィールドで使用できなくなることがあります。

_最初に報告されたのは 2025年1月10日（PT）です。_
