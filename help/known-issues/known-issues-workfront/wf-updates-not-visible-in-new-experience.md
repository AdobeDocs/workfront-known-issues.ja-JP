# 更新：新しいコメントエクスペリエンスでは一部の更新が欠落している

>[!NOTE]
>
>この問題は、2023年12月21日に修正されました。

ユーザーが新しいコメントエクスペリエンスで更新を表示する際、表示されるはずのコメントの一部が表示されません。ユーザーが従来のコメントエクスペリエンスに切り替えると、すべてのコメントが表示されます。

_最初に報告されたのは 2023年11月28日（PT）です。_
