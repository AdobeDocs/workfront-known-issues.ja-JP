# 更新：新しいコメントエクスペリエンスに一部の更新がありません

ユーザーが新しいコメントエクスペリエンスで更新を表示している場合、表示する必要のあるコメントの一部は表示されません。 ユーザーが従来のコメント作成エクスペリエンスに切り替えた場合は、すべてのコメントが表示されます。

_最初に報告されたのは 2023年11月28日（PT）です。_
