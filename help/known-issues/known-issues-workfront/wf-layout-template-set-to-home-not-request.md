# レイアウトテンプレート：ランディングページが正しくありません

要求者ライセンスを持つユーザーが、要求ページをランディングページとして設定したレイアウトテンプレートに割り当てられると、そのユーザーは代わりに、ランディングページとしてホームページに移動します。

これは要求元のライセンスに関して報告されています。

_最初に報告されたのは 2024年3月22日（PT）です。_
