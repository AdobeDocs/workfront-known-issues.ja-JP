# [!UICONTROL ワークロードバランサー]: [!UICONTROL 合計時間] の合計と一致しない [!UICONTROL 予定時間]

ユーザーが [!UICONTROL ワークロードバランサー] を増やし、 [!UICONTROL 予定時間]の場合、合計は [!UICONTROL 合計時間] をクリックします。 タイムラインの再計算では問題は解決しません。

_最初に報告されたのは 2023年6月8日（PT）です。_
