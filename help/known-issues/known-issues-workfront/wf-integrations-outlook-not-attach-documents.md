# 統合：からのリクエストを作成する際に添付されないドキュメント [!DNL Outlook]

ユーザーが [!DNL Outlook]を指定した場合、電子メールに添付されたドキュメントは要求に添付されません。

これは、次の添付ファイルタイプに関して報告されています。

XLSPDF

_最初に報告されたのは 2024年2月5日（PT）です。_
