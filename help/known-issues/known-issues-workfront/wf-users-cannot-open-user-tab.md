# ユーザー：ユーザーページを開くと、エラーが発生する

>[!NOTE]
>
>この問題は、2025年12月12日（PT）に修正されました。

ユーザーがメインメニューからユーザーページを開こうとすると、ページが開かず、次のエラーが表示されます。

「エラーが発生したため、問題を解決中です。 作業を続行するには、このブラウザーページを更新してみてください。」が表示されます。

_最初に報告されたのは、2025年12月8日（PT）です。_
