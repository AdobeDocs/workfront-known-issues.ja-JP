# ドキュメント：フォルダーからルートにドキュメントを移動できません

ユーザーが作成したフォルダーからプロジェクトのドキュメントルートフォルダーにフォルダーを移動しようとすると、ドキュメントは移動されず、次のエラーが表示されます。

&quot;[!UICONTROL destinationFolderD を null にすることはできません]&quot;
