# リクエスト：リクエストキューが複数回表示される

>[!NOTE]
>
>この問題は、従来のレポートエクスペリエンスで報告されています。

ユーザーがリクエストの作成を開始し、リクエストキューを選択すると、同じリクエストキューがドロップダウンリストに複数回表示されます。

_最初に報告されたのは 2025年12月30日（PT）です。_
