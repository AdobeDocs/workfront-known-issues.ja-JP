# ワークロードバランサー：ページの更新後に手動の時間変更が正確に行われない

ユーザーが手動で時間をワークロードバランサーに入力し、ページが更新されると、表示される時間が不正確になります。

_最初に報告されたのは 2023年6月8日（PT）です。_
