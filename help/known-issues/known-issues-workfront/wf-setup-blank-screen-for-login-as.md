# 設定：「別名でログイン」すると空白の画面に誘導される

管理者が別のユーザーとしてログインすると、そのユーザーのアカウントが表示されるのではなく、空白の画面が表示されます。

**回避策**

画面を更新します。 これには、数回の試行が必要な場合があります。

_最初に報告されたのは 2024年7月17日（PT）です。_
