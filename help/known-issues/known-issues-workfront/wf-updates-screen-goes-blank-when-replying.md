# 更新：別の会社からユーザーに返信する際に、画面が空白になる

ユーザーが別の会社のユーザーからコメントに返信しようとすると、画面が空白になります。

これは、ユーザーが他の会社のユーザーを表示する権限を持っていないためです。

_最初に報告されたのは 2024年1月8日（PT）です。_
