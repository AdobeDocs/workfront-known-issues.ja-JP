# 更新：API を通じて作成されたコメントがサンドボックス環境に表示されない

Workfront API を使用してコメントを作成すると、そのコメントが実稼動環境に表示されます。 ただし、コメントはサンドボックス環境には表示されません。

_最初に報告されたのは 2024年4月8日（PT）です。_
