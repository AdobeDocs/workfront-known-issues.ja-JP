# [!UICONTROL ホーム]: [!UICONTROL プロジェクト名] は、 [!UICONTROL 作業用リスト]

>[!NOTE]
>
>製品チームは現在、この問題の解決を評価中です。 この問題が解決されると、製品のお知らせに記載され、メンテナンスのアップデートでは表示されません。

配達確認の承認を求められたユーザーにはアクセス権がなく、ユーザーの [!UICONTROL 作業用リスト] in [!UICONTROL ホーム] 並べ替え基準： [!UICONTROL プロジェクト名]に設定されていない場合、ユーザーはプロジェクト名を表示できず、代わりに GUID が表示されます。

_最初のレポートは 2021 年 1 月 5 日です。_
