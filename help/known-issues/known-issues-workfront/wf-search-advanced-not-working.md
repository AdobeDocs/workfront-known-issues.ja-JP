# 検索：詳細検索で期待した結果が返されない

>[!NOTE]
>
>この問題は 2025年6月2日（PT）に修正されました。

ユーザーが詳細検索を使用したときに、期待した結果が返されません。この問題は、詳細検索にフィルターが適用されていない場合でも発生する可能性があります。

**回避策**

基本検索は、期待どおりに機能します。

_最初に報告されたのは 2025年5月27日（PT）です。_
