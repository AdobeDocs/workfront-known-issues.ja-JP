# 報告書：スクロール時にマトリックスレポートが正しく表示されない

>[!NOTE]
>
>この記事は 2022年8月18日に修正されました。

ユーザーがマトリックスレポートを閲覧中にスクロールすると、レポートの一部の視覚的要素が重なったり重複したりすることがあります。

_最初に報告されたのは 2022年6月3日です。_
