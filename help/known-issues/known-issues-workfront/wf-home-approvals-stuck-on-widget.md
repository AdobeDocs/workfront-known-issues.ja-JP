# ホーム：承認が停止しています [!UICONTROL 承認待ち] widget

<!--on WF and WFP TOCs-->

ドキュメントの新しいバージョンがアップロードされ、前のバージョンが未完了の承認を持つと、ドキュメントの古いバージョンが承認者の [!UICONTROL 承認待ち] ウィジェット。 新しいバージョンがあるので、承認できません。また、ウィジェットから削除することはできません。

_最初に報告されたのは 2023年9月25日（PT）です。_
