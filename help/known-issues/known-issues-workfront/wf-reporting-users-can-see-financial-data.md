# レポート：ユーザーは、キャンバスダッシュボードのビジュアライゼーションで財務データを表示できます

アクセスレベルで財務データへの表示または編集アクセス権を持つユーザーは、財務表示権限がタスクレベルまたはプロジェクトレベルで削除されている場合でも、キャンバスダッシュボードのビジュアライゼーションで財務データを表示できます。

アクセスレベルで財務データ権限を持たないユーザーには、レポートに財務データが表示されません。

財務データを表示するユーザーは、既に表示権限を持っているレコード（プロジェクト、タスク、イシューなど）に制限されます。 アクセスできないレコードの財務値は表示されません。

レポート作成者は、ダッシュボードに財務データを含める際には注意を払い、意図しないアクセスを防ぐためにダッシュボードを共有する相手に注意する必要があります。
