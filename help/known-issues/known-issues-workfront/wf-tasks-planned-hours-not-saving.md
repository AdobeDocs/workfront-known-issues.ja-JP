# タスク：タスクの作成時に予定時間数が保存されない

>[!NOTE]
>
>この問題は、2025年12月3日（PT）に修正されました。

ユーザーがタスクを作成して予定時間数を入力してからタスクを保存すると、予定時間数が保存されません。

**回避策**

タスクを作成してから、予定時間数を追加するよう編集します。

_最初に報告されたのは、2025年10月10日（PT）です。_
