# タスク：タスクの作成時に予定時間数が保存されない

>[!NOTE]
>
>この問題は、2025年12月3日に修正されました。

ユーザーがタスクを作成して予定時間数を入力してからタスクを保存すると、予定時間数は保存されません。

**回避策**

タスクを作成してから編集し、予定時間数を追加します。

_最初に報告されたのは 2025年10月10日（PT）です。_
