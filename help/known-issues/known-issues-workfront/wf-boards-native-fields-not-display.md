# ボード：Workfront のネイティブフィールドがカードに表示されない

>[!NOTE]
>
>この問題は、2024年5月2日（PT）に修正されました。

ネイティブ（非カスタム）フィールドを表示するように設定されたカードをユーザーが表示した場合、これらのフィールドはカードに表示されません。

**回避策**

カードをクリックして、カードが表すオブジェクトに移動します。

_最初に報告されたのは 2024年4月15日（PT）です。_
