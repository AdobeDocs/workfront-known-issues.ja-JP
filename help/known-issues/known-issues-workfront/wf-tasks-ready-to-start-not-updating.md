# タスク：先行タスクが完了しても、開始準備完了フィールドが更新されない

先行タスクを含むタスクの「開始準備完了」フィールドが、必要なすべての先行タスクが「完了」とマークされているにもかかわらず、True に更新されない。

_最初に報告されたのは 2024年10月1日（PT）です。_
