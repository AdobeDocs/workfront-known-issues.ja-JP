# タスク：先行タスクが完了しても、「開始準備完了」フィールドが更新されない

>[!NOTE]
>
>この問題は、2024年12月9日に修正されました。

必要なすべての先行タスクが「完了」とマークされているにもかかわらず、先行タスクを含むタスクの「開始準備完了」フィールドが True に更新されません。

_最初に報告されたのは 2024年10月1日（PT）です。_
