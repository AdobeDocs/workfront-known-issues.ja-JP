# ボード：タグの色を編集できません

ユーザーが [!UICONTROL タグマネージャー]の場合、ユーザーはドロップダウンから別の色を選択できません。 ユーザが色をクリックした場合、クリックは無効になります。

この問題は、 [!UICONTROL タグマネージャー]. 既存のタグ（色を含む）は、通常どおりカードに割り当てることができます。

_最初に報告されたのは 2023年10月30日（PT）です。_
