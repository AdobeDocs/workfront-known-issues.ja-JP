# ステータス：ステータスを非表示にする際にエラーが発生する

ユーザーがステータスを非表示にしようとすると、ステータスは非表示にならず、次のエラーが表示されます。

「[!UICONTROL エラーが発生しました。何が問題だったのかを把握して解決するために、Workfront 担当者にお問い合わせください。]」
_最初に報告されたのは 2023年11月30日（PT）です。_
