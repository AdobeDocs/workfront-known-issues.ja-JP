# タイムシート：&quot;[!UICONTROL 最近の変更は保存されませんでした]&#39;&#39;タイムシートの変更を自動保存中にエラーが発生しました

「Y[!UICONTROL 最近の変更は保存されませんでした。 表示するページを更新します。]&quot;

これは、次の項目を編集する際に報告されています。

* 時間
* 件のタスク
