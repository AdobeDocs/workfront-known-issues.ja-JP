# 通知：ユーザーが予期された通知を受け取らない

ユーザーに通知を送信する必要があるイベントが発生しても、そのユーザーは通知を受信しません。 この問題は、ユーザー通知設定によって通知が許可される場合でも発生する可能性があります。

この問題は、アプリ内通知とメール通知の両方に影響します。

_最初に報告されたのは、2026年2月12日（PT）です。_
