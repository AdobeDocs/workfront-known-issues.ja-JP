# 通知：ユーザーが期待される通知を受信しない

ユーザーに通知を送信する必要があるイベントが発生した場合、そのユーザーは通知を受け取りません。 ユーザー通知設定で通知が許可される場合でも、この問題が発生する可能性があります。

この問題は、アプリ内とメールの両方の通知に影響します。

_最初に報告されたのは2026年2月12日です。_
