# レイアウトテンプレート：削除されたビューを引き続き使用できます

管理者がレイアウトテンプレートを使用してリストからビューを削除しても、そのビューはユーザーが使用できるビューのリストに表示されます。 選択した場合、ビューは適用されないか、データを含まないビューの列が適用されます。

_最初に報告されたのは 2025年1月21日（PT）です。_
