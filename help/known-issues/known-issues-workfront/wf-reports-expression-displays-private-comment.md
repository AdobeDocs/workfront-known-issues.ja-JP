# レポート：値式フィールドにプライベート更新が表示されます

レポートフィールドに非公開の更新を参照する値式が含まれている場合、非公開の更新に含まれていないユーザーは、その式をレポートで表示できます。

_最初に報告されたのは 2023年9月26日（PT）です。_
