# プルーフ：プレビュー環境でプルーフを作成できない

>[!NOTE]
>
>このイシューは再現できなかったため、クローズされました。

ユーザーがプレビューサンドボックス環境でプルーフを作成しようとすると、プルーフは作成されず、「生成できませんでした」というメッセージがユーザーに表示されます。

_最初に報告されたのは 2024年10月17日（PT）です。_
