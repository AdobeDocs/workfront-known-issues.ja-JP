# プルーフ：プレビュー環境でプルーフを作成できない

>[!NOTE]
>
>この問題は、プレビューサンドボックス環境でのみ報告されています。

ユーザーがプレビューサンドボックス環境でプルーフを作成しようとすると、プルーフは作成されず、「生成に失敗しました」というメッセージが表示されます。

_最初に報告されたのは 2024年10月17日（PT）です。_
