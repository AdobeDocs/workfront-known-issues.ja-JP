# ホーム：承認待ちウィジェットに「承認」ボタンが表示されない

<!--
>[!NOTE]
>
>This issue was fixed on February 13, 2025.
-->

ホームで承認待ちウィジェットを使用する場合、承認決定を選択するボタンが表示されません。

**回避策**

承認決定は、承認に直接移動することで行うことができます。

_最初に報告されたのは 2024年11月21日（PT）です。_
