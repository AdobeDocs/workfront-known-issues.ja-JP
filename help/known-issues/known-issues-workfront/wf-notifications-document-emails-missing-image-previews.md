# 通知：ドキュメント通知メールで画像のプレビューを表示できない

>[!NOTE]
>
>この問題は、Gmail の制限により発生し、修正される予定はありません。

Gmail を使用してドキュメントのレビューまたは承認メールを開くと、画像のプレビューが表示されません。

**回避策**

Gmail 以外のメールアプリケーションを使用すると、画像のプレビューが正しく表示されます。

_最初に報告されたのは 2024年10月17日（PT）です。_
