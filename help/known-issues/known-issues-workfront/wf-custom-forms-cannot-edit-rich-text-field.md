# カスタムフォーム：リッチテキストフィールドを編集できない

>[!NOTE]
>
>この問題は、2025年12月22日（PT）に修正されました。

ユーザーがカスタムフォームでリッチテキストフィールドを編集しようとすると、変更が保存されず、フィールドに変更が反映されません。

_最初に報告されたのは、2025年12月3日（PT）です。_
