# カスタムフォーム：リッチテキストフィールドを編集できない

ユーザーがカスタムフォームでリッチテキストフィールドを編集しようとしても、変更が保存されず、フィールドが更新されて変更が反映されません。

_最初に報告されたのは 2025年12月3日（PT）です。_
