# ドキュメント：EPS ファイルが現在サポートされていない

<!--WF, WFP-->

>[!NOTE]
>
>この問題は 2023年7月27日（PT）に修正されました。

Workfront は現在、EPS ファイルからのプルーフの作成をサポートしていません。

**回避策**

ファイルを PDF として書き出し、その PDF を Workfront にアップロードします。その後、PDF からプルーフを作成できます。

_最初に報告されたのは 2023年7月19日（PT）です。_
