# ドキュメント： EPSファイルは現在サポートされていません

<!--WF, WFP-->

>[!NOTE]
>
>この問題は 2023年7月27日（PT）に修正されました。

現在、WorkfrontはEPSファイルからの配達確認の作成をサポートしていません。

**回避策**

ファイルをPDFとして書き出し、PDFをWorkfrontにアップロードします。 その後、配達確認を「 」PDFから作成できます。

_最初に報告されたのは 2023年7月19日です。_
