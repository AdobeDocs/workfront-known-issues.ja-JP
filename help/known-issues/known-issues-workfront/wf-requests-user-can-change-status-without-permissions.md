# リクエスト：権限がない場合でも、ユーザーがステータスを変更できる

投稿アクセス権を持ち、イシューのステータスを変更する権限を持たないユーザーは、ステータスを変更できます。

_最初に報告されたのは 2024年7月29日（PT）です。_
