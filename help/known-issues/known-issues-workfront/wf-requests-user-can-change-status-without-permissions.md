# リクエスト：権限がないユーザーでもステータスを変更できる

>[!NOTE]
>
>この問題は、設計どおりの動作であるため、クローズされました。

参加アクセス権を持ち、イシューのステータスを変更する権限を持たないユーザーは、ステータスを変更できます。

_最初に報告されたのは 2024年7月29日（PT）です。_
