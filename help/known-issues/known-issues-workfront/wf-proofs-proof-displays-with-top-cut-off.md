# 配達確認：配達確認はトップカットオフで表示されます

<!--WF and WFP TOCs-->

ユーザーが校正ビューアで配達確認を表示すると、配達確認の上部が切り取られているように見えます。 ユーザーが配達確認をダウンロードすると、何も切り取らずに配達確認全体が表示されます。

_最初に報告されたのは 2023年7月28日（PT）です。_
