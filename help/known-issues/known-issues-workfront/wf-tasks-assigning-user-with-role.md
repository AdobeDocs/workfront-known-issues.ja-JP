# タスク：ユーザーをタスクに割り当てると、担当業務がリストから表示されなくなる

>[!NOTE]
>
>この問題は 2024年7月4日（PT）に修正されました。

担当業務をタスクに割り当て、そのタスクを担当業務を持つユーザーに割り当てると、担当業務が割り当てのリストから表示されなくなります。

_最初に報告されたのは 2024年5月21日（PT）です。_
