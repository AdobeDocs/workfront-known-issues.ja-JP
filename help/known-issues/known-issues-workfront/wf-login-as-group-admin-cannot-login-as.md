# 別名でログイン：グループ管理者が別のユーザーとしてログインできない

>[!NOTE]
>
>この問題は、2025年3月27日（PT）に修正されました。

グループ管理者が別のユーザーとしてログインしようとすると、そのユーザーとしてログインされません。 代わりに、「別名でログイン済み」バナーにグループ管理者の名前が表示され、グループ管理者には自分のアカウントが表示されます。

_最初に報告されたのは、2025年2月18日（PT）です。_
