# 別名でログイン：グループ管理者は別のユーザーとしてログインできません

グループ管理者が別のユーザーとしてログインしようとしても、ユーザーとしてログインしません。 代わりに、「別名でログイン」バナーにグループ管理者の名前が表示され、グループ管理者には自分のアカウントが表示されます。

_最初に報告されたのは 2025年2月18日（PT）です。_
