# ユーザー：一括割り当てを行う際にユーザーが見つからない

ユーザーが一括割り当てを試み、担当業務がタスクに割り当てられると、その担当業務を持つユーザーはリストに表示されず、割り当てユーザーは選択できません。

この動作は、ワークロードバランサーでも報告されています。

_最初に報告されたのは 2024年5月17日（PT）です。_
