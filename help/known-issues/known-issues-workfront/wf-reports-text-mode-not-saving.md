# レポート：テキストモードで行った変更は、レポートまたはビューには適用されません

ユーザーがレポートまたはビューに対してテキストモードで変更を行った場合、その変更は保存されず、レポートまたはビューに適用されません。

_最初に報告されたのは 2024年2月21日（PT）です。_
