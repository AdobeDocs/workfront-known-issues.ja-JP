# ログ時間：ユーザーは現在の日の時刻をログに記録できません

ユーザーが [ 更新 ] 領域で時間をログに記録しようとすると、現在の日が灰色表示になり、現在の日の時間をログに記録できなくなります。

_最初に報告されたのは 2024年3月4日（PT）です。_
