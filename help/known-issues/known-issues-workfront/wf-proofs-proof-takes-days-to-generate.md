# プルーフ：プルーフの生成に過度の時間がかかる

>[!NOTE]
>
>この問題は 2023年9月21日（PT）に修正されました。

ユーザーがプルーフを生成しようとすると、プルーフの生成に長い時間がかかります。プルーフの生成に数日かかる場合があります。

_最初に報告されたのは 2023年8月2日（PT）です。_
