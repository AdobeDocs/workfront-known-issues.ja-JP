# プルーフ：静的プルーフのテキストの書式が web サイトの書式と一致しない

<!--WF, WFP TOCs-->

ユーザーが web サイトから静的プルーフを作成する際、web サイト上で太字になっているテキストが、プルーフ上では太字として表示されません。

_最初に報告されたのは 2023年8月23日（PT）です。_
