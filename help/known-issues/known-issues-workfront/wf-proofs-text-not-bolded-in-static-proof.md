# 配達確認：静的配達確認のテキスト書式が Web サイトの書式と一致しません

<!--WF, WFP TOCs-->

ユーザーが Web サイトから静的配達確認を作成すると、Web サイト上で太字のテキストは配達確認で太字には表示されません。

_最初に報告されたのは 2023年8月23日（PT）です。_
