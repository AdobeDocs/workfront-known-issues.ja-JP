# モバイル：ログアウトできず、ユーザープロファイル情報が見つからない

>[!NOTE]
>
>この問題は 2024年9月19日（PT）に修正されました。

モバイルアプリケーションのログアウトオプションでは、一部のユーザーは正常にログアウトできません。ユーザープロファイルページも空白で表示され、エラーメッセージが生成されます。

_最初に報告されたのは 2024年7月18日（PT）です。_
