# プルーフ：アクティブなユーザーが非アクティブ化されたように表示される

アクティブであるにもかかわらず、ユーザーは一部のプルーフでディアクティベートされているように表示され、アクセスできません。

_最初に報告されたのは 2024年9月6日（PT）です。_
