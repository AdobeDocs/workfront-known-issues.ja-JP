# 問題：問題のステータスがドロップダウンに表示されない

>[!NOTE]
>
>このイシューは 2024年3月14日（PT）に修正されました。

ユーザーが問題のヘッダーで問題のステータスを変更しようとする際に、使用可能なすべての問題のステータスがドロップダウンに表示されません。

この問題は、「[!UICONTROL 処理中]」ステータスとして報告されています。

_最初に報告されたのは 2023年11月6日（PT）です。_
