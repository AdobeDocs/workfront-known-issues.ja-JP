# グループ：サブグループを追加できない

>[!NOTE]
>
>この問題は 2024年6月13日（PT）に修正されました。

ユーザーが既存のサブグループをグループに追加しようとすると、「保存」ボタンは無効になり、サブグループは追加されません。

_最初に報告されたのは 2024年5月1日（PT）です。_
