# アップデート：ユーザーの代わりに行ったアップデートがユーザーから直接行ったと表示される

>[!NOTE]
>
>このイシューは 2023年3月30日（PT）に修正されました。

管理者がユーザーとしてログインしてコメントを作成すると、そのコメントは、ユーザーの代わりに管理者が行ったものではなく、ユーザーが直接として表示されます。

_最初に報告されたのは 2023年3月1日（PT）です。_
