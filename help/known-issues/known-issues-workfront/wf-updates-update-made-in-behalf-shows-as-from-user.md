# 更新：ユーザーに代わっておこなわれた更新は、ユーザーから直接「 」として表示されます

管理者がユーザーとしてログインしてコメントを作成すると、そのコメントは、ユーザーの代わりに管理者から直接表示されるのではなく、ユーザーから直接表示されます。

_最初に報告されたのは 2023年3月1日（PT）です。_
