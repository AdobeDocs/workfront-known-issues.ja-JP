# プロジェクト：問題がプロジェクトに変換されると、カスタムフィールドが見つかりません

>[!NOTE]
>
>この問題は 2023年3月16日に修正されました。

ユーザーがイシューをテンプレートを使用してプロジェクトに変換すると、イシューに含まれていたカスタムフィールドはプロジェクトに表示されません。 この問題が影響するのは管理者以外のユーザーのみです。

_最初に報告されたのは 2023年2月6日です。_
