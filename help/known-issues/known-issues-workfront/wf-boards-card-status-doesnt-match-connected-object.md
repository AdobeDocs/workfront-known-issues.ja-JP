# ボード：カードのステータスが接続オブジェクトと一致せず、フィルターをバイパスする

動的ボードの取り込み列にあるカードに間違ったステータス情報が表示され、そのステータスに基づいて適切にフィルタリングされない。

_最初に報告されたのは 2024年8月8日（PT）です。_
