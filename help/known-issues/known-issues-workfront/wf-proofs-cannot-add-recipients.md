# プルーフ：プルーフに受信者を追加できない

>[!NOTE]
>
>この問題は、2024年12月13日に修正されました。

ユーザーが共有／受信者を追加オプションを使用してプルーフにユーザーを追加しようとすると、ユーザーが名前をクリックしても受信者は追加されません。

**回避策**

別の方法を使用してプルーフを共有します。

_最初に報告されたのは 2024年6月1日（PT）です。_
