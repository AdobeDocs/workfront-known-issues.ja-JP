# ユーザー：ユーザープロファイルページが連続して読み込まれる

>[!NOTE]
>
>この問題は 2024年7月18日（PT）に修正されました。

ユーザーがユーザープロファイルページを表示しようとすると、ページは読み込み中のように見えますが、完全には読み込まれません。

この問題は、[!UICONTROL  別名でログイン ] 機能を使用している場合に報告されています。

_最初に報告されたのは 2024年7月8日（PT）です。_
