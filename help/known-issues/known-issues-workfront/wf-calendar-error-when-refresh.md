# カレンダー：カレンダーの更新中にエラーが表示される

ユーザーがカレンダーを表示し、ページを更新する際、「Whoops」エラーが表示されます。カレンダー上のデータは期待どおりに表示されますが、エラーメッセージによって見えにくくなる場合があります。

_最初に報告されたのは 2024年3月13日（PT）です。_
