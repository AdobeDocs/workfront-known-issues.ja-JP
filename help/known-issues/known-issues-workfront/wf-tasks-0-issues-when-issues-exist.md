# 問題：問題が存在する場合、左側のナビゲーションに「(0)」の問題が表示されます

タスクが存在するタスクをユーザーが表示している場合、タスクの左側のナビゲーションに問題がないことを示す「(0)」が「問題」領域に表示されます。 タスクの [ タスク ] 領域を表示した場合は、タスクの [ タスク ] に問題が表示されます。

_最初に報告されたのは 2022年10月17日です。_
