# ドキュメント：ドキュメント数が不正確です

ドキュメントが添付されたオブジェクトをユーザーが表示すると、左側のナビゲーションの「ドキュメント」タブに表示されるドキュメントの数が不正確になる場合があります。 表示される数値は、ルートフォルダー内のドキュメントの数です。 サブフォルダー内のドキュメントはカウントされません。

_最初に報告されたのは 2024年6月20日（PT）です。_
