# ドキュメント：ドキュメント数が不正確である

ユーザーが閲覧中のオブジェクトにドキュメントが添付されている場合、左側のナビゲーションの「ドキュメント」タブに不正確なドキュメント数が表示されることがあります。表示される数はルートフォルダー内のドキュメント数です。サブフォルダー内のドキュメントはカウントされません。

_最初に報告されたのは 2024年6月20日（PT）です。_
