# カレンダー：タスクが表示されない

>[!NOTE]
>
>製品チームは現在、この問題の解決策を評価中です。この問題が解決されると、メンテナンス更新ではなく、製品のお知らせで通達されます。

ユーザーがリストレポートとカレンダーレポートで同じレポートを使用している場合、一部のタスクがリストレポートに表示され一方で、カレンダーレポートには表示されないことがあります。

_最初に報告されたのは 2019年10月31日です。_
