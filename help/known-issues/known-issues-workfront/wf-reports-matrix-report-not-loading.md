# レポート：マトリックスレポートを読み込む際の問題

>[!NOTE]
>
>この問題は 2023年8月31日（PT）に修正されました。

マトリックスレポートをグラフとして読み込もうとすると、次のいずれかが発生することがあります。

* レポートの一部の情報が読み込まれない
* レポートには「[!UICONTROL サーバーからコンテンツを読み込めません]」というエラーが表示される」

**回避策**

レポートの詳細を表示します。

_最初に報告されたのは 2023年7月24日（PT）です。_
