# プロジェクト：プログラムにプロジェクトを追加する際にエラーが発生する

>[!NOTE]
>
>この問題は 2025年8月21日（PT）に修正されました。

ユーザーがプログラムに既存のプロジェクトを追加しようとしても、プロジェクトが追加されず、次のエラーメッセージが表示されます。

「セカンダリコンテンツの読み込み中にエラーが発生しました...」

_最初に報告されたのは 2025年7月4日（PT）です。_
