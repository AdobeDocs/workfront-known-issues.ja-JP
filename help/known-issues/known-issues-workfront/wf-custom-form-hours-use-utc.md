# カスタムフォーム： [!UICONTROL 時間] 計算フィールドの関数は UTC を使用します

計算フィールドに [!UICONTROL 時間] 関数の値は、期待されるタイムゾーンではなく、UTC に基づいて返されます。 したがって、HOUR 値に基づく計算は正しくありません。

_最初に報告されたのは 2022年10月17日です。_
