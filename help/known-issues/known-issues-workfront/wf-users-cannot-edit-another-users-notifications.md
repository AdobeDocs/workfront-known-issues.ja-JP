# ユーザー：別のユーザーの通知設定を編集できない

>[!NOTE]
>
>この問題は 2025年9月25日（PT）に修正されました。

ユーザーが別のユーザーの通知設定を編集しようとすると、設定を編集できず、次のメッセージが表示されます。

「財務データを編集するための十分なアクセス権限がありません。」

_最初に報告されたのは 2025年9月9日（PT）です。_
