# タスク：タスクの作成時にユーザーの役割を選択できない

ユーザーがタスクを作成中にユーザーの役割を割り当てようとすると、役割を割り当てるオプションが表示されず、ユーザーはユーザーの役割を割り当てることができません。

**回避策**

タスクを作成し、タスクを編集して詳細割り当てでユーザーの役割を割り当てます。

_最初に報告されたのは 2025年11月28日（PT）です。_
