# タスク：タスクの作成時にユーザーの役割を選択できない

>[!NOTE]
>
>この問題は、2025年12月21日（PT）に修正されました。

ユーザーがタスクの作成中にユーザーの役割を割り当てようとすると、役割を割り当てるオプションが表示されず、ユーザーはユーザーの役割を割り当てることができません。

**回避策**

タスクを作成してから、「高度な割り当て」でユーザーの役割を割り当てるよう編集します。

_最初に報告されたのは、2025年11月28日（PT）です。_
