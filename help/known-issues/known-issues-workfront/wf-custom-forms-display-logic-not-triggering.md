# カスタムフォーム：リクエストフォームで表示ロジックが正しく表示されない

>[!NOTE]
>
>この問題は2026年2月13日（PT）に修正されました。

ユーザーがリクエストに接続されたカスタムフォームに入力する際、表示ロジックに従わず、一部のフィールドが期待どおりに表示されない場合があります。

**回避策**

リクエストドラフトを閉じてから、再度開きます。

_最初に報告されたのは2026年1月15日です。_
