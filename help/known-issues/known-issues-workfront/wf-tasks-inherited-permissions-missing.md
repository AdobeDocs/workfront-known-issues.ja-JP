# タスク：タスクに継承された権限が欠落している

>[!NOTE]
>
>この問題は 2024年7月26日（PT）に修正されました。

タスクに対する権限を継承する必要があったユーザーには、その権限はありません。

_最初に報告されたのは 2024年7月19日（PT）です。_
