# プルーフ：プルーフの生成が遅い

>[!NOTE]
>
>この問題は 2025年3月19日（PT）に修正されました。

ユーザーが Workfront でプルーフを作成する際、プルーフの作成に 3～10 分かかる場合があります。プルーフは生成されますが、時間がかかります。

_最初に報告されたのは 2024年1月21日（PT）です。_
