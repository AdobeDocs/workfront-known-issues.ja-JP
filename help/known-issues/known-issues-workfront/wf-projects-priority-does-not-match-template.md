# プロジェクト：テンプレートを使用して作成されたプロジェクトがテンプレートの優先度と一致しない

ユーザーがテンプレートを使用して要求をプロジェクトに変換すると、新しく作成されたプロジェクトの優先度がテンプレートまたは要求と一致しません。

_最初に報告されたのは 2024年5月2日（PT）です。_
