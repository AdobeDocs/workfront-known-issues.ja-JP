# リスト：リストにすべての項目が表示されない

ユーザーがオブジェクトリストを表示し、（リストがページ分割されないように）右下隅の「すべて」を選択すると、項目はページ分割されたままになり、すべてのオブジェクトがページに表示されません。

_最初に報告されたのは 2024年4月11日（PT）です。_
