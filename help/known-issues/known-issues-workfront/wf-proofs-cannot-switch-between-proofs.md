# プルーフ：[!UICONTROL プルーフビューア]でプルーフを切り替えることができない

ユーザーが[!UICONTROL プルーフビューア]でプルーフを表示している際、別のプルーフに切り替えると、「プルーフを切り替え」ボタンが応答しなくなります。ユーザーは、表示していた元のプルーフや別のプルーフに切り替えることはできません。

_最初に報告されたのは 2023年3月31日（PT）です。_
