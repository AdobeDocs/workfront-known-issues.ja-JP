# タスク：タスク期間のインライン編集時に[!UICONTROL 経過時間]を選択できない

>[!NOTE]
>
>この問題は 2022年7月21日に修正されました。

ユーザーがタスクリストを表示中に[!UICONTROL タスク期間]を編集しようとすると、次の期間単位を使用できません。

* [!UICONTROL 経過時間数 (分)]
* [!UICONTROL 経過時間数]
* [!UICONTROL 経過日数]
* [!UICONTROL 経過週数]
* [!UICONTROL 経過月数]

_最初に報告されたのは 2022年7月1日です。_
