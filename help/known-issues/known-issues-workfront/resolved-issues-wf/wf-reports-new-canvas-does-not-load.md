# 報告書：[!UICONTROL 新しいレポートキャンバス]が読み込まれない

>[!NOTE]
>
>この問題は 2022年7月25日に解決されました。

ユーザーが[!UICONTROL 新しいレポートキャンバス]を読み込もうとすると、「[!UICONTROL データを読み込み中]」画面が表示されますが、[!UICONTROL 新しいレポートキャンバス]は読み込まれません。

_最初に報告されたのは 2022年7月4日です。_
