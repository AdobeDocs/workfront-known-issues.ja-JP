# 更新：ユーザーにタグ付けし、別の場所に移動すると、タグ付けされたユーザーが削除される

>[!NOTE]
>
>この問題は 2022年8月4日に修正されました。

ユーザーが更新ストリームで通知するために別のユーザーを入力した後、別のタブに移動して更新ストリームに戻ると、タグ付けされたユーザーは削除されます。

_最初に報告されたのは 2022年7月7日です。_
