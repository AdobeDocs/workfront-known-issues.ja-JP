# タイムシート：タイムシート作成時の問題

>[!NOTE]
>
>この問題は 2022年7月28日に修正されました。

タイムシートの作成に関して次のような問題が報告されています。

* ユーザーがロールのタイムシートを作成しようとすると、タイムシートは作成されず、「[!UICONTROL プライマリキー値「XXXXXXXXXXX」を持つユーザーが見つかりませんでした]」というエラーが表示されます。
* ユーザーがチームのタイムシートを作成しようとすると、[!UICONTROL typeahead] フィールドにチームが生成されず、「[!UICONTROL タイムシートを作成]」ボタンが無効になります。

_最初に報告されたのは 2022年6月22日です。_
