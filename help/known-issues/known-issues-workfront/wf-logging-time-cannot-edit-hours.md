# 時間の記録：タスクやイシューに関する時間を編集できない

>[!NOTE]
>
>この問題は、2024年5月2日（PT）に修正されました。

ユーザーがタスクやイシューの時間を編集しようとすると、変更が保存されません。

_最初に報告されたのは 2024年4月1日（PT）です。_
