# リクエスト：リクエストページ（「../submitted」URL）のダッシュボードにアクセスできない

>[!NOTE]
>
>この問題は、2024年5月16日（PT）に修正されました。

リクエストページでダッシュボードにアクセスしようとすると、URL に一時的にダッシュボードの URL が表示され、その後すぐに送信された URL（/submitted）に切り替わります。

_最初に報告されたのは 2024年5月16日（PT）です。_
