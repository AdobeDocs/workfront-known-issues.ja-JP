# 要求：要求ページのダッシュボードにアクセスできない（「。./submitted」 URL）

>[!NOTE]
>
>この問題は、2024年5月16日（PT）に修正されました。

要求ページ上のダッシュボードにアクセスしようとすると、その URL が一時的にダッシュボードの URL を表示した後、送信された URL （/submitted）にすばやく切り替わります。

_最初に報告されたのは 2024年5月16日（PT）です。_
