# レイアウトテンプレート：レイアウトテンプレートを使用してタスクの概要に追加した際に、カスタムデータフィールドが表示されない

管理者がレイアウトテンプレートを使用してタスクの「概要」セクションにカスタムデータフィールドを追加すると、タスクの「概要」セクションを見ているユーザーには、フィールドが空として表示されます。

_最初に報告されたのは 2024年10月2日（PT）です。_
