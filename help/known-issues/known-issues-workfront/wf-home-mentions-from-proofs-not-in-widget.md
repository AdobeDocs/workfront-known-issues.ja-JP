# ホーム：プルーフからのメンションがメンションウィジェットに表示されない

>[!NOTE]
>
>この問題は2026年2月19日（PT）に修正されました。

プルーフに関するコメントでユーザーがメンションされた場合、そのメンションはホームのメンションウィジェットには含まれません。

_最初に報告されたのは2026年1月23日です。_
