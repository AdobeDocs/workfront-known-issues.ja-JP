# 統合：統合シェル上の組織で JumpSeat 統合が失敗する

>[!NOTE]
>
>この問題は解決されましたが、以下の **回避策** に示すように JumpSeat を再設定する必要があります。

ユーザーの組織がAdobeの UnifiedBecky シェルに移行されると、既存の JumpSeat 統合が機能しません。

**回避策**

JumpSeat 管理者パネルで、アプリケーション URL を `workfront.com` から `workfront.adobe.com` に変更します。

_最初に報告されたのは 2024年10月30日（PT）です。_
