# 通知：更新のメール通知が送信されない

>[!NOTE]
>
>このイシューは再現できなかったため、クローズされました。

オブジェクトの更新に対するメール通知を有効にしているユーザーに、期待したメール通知が届かないことがあります。

_最初に報告されたのは 2024年10月1日（PT）です。_
