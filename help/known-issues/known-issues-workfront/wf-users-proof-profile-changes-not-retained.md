# ユーザー：プルーフプロファイルへの変更に関する問題

>[!NOTE]
>
>この問題は、2025年12月17日（PT）に修正されました。

ユーザーのプルーフプロファイルへの変更に関して、次の問題が報告されています。

* ユーザーが別のプルーフプロファイルをあるユーザーに割り当てて保存すると、変更が保存されず、そのユーザーは以前のプルーフプロファイルを保持します。
* 正しいプルーフプロファイルが割り当てられていても、ユーザーのプルーフプロファイルが間違ったプルーフプロファイルとして表示される場合があります。
* ユーザーに対して他の変更が行われると、プルーフの権限も変更される場合があります。

_最初に報告されたのは、2025年11月24日（PT）です。_
