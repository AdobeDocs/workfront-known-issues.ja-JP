# ユーザー：プルーフプロファイルに対する変更に関する問題

>[!NOTE]
>
>この問題は、2025年12月17日に修正されました。

ユーザーのプルーフプロファイルに変更を加えることに関して、次の問題が報告されています。

* ユーザーが別のプルーフプロファイルをユーザーに割り当てて保存すると、変更は保存されず、ユーザーは以前のプルーフプロファイルを保持します。
* 正しいプルーフプロファイルが割り当てられていても、ユーザーのプルーフプロファイルが間違ったプルーフプロファイルとして表示される場合があります。
* ユーザーに対してその他の変更が行われると、プルーフの権限も変更される場合があります。

_最初に報告されたのは 2025年11月24日（PT）です。_
