# プロジェクト／イシュー：プロジェクトでイシューが正しく解決されない

>[!NOTE]
>
>この問題は 2023年8月3日に修正されました。

ユーザーがイシューの解決オブジェクトであるプロジェクトのステータスを変更すると、イシューのステータスはプロジェクト上のステータスと同じキーに一致しないステータスに変更されます。

**回避策**

解決関係を削除し、ステータスを個別に設定します。

_最初に報告されたのは 2023年7月17日（PT）です。_
