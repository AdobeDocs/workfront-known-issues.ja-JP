# Workfront Planning:「リクエストを送信できませんでした」エラー

ユーザーがリクエストを送信すると、「リクエストの送信に失敗しました」というメッセージが表示されます。 ただし、リクエストは正しく送信されます。

_最初に報告されたのは、2026年2月11日（PT）です。_
