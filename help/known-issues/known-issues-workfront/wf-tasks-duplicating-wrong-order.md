# タスク：タスクを作成または複製すると、タスクリストが間違った順序で表示される

ユーザーがタスクリストでタスクを作成または複製すると、タスクリストにはタスクの順序番号に従って、間違った順序でタスクが表示されます。タスクには正しく番号が付けられていますが、表示される順番が正しくありません。

**回避策**

ページを更新します。

_最初に報告されたのは 2023年11月27日（PT）です。_
