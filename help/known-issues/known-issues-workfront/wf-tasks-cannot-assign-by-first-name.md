# タスク：タスク割り当てで名を検索してもユーザーが返されない

ユーザーがタスクの割り当てを行っており、担当者を名で検索しようとすると、検索でユーザーが返されず、「結果が見つかりません」というメッセージが表示されます。

**回避策**

フルネームで検索すると、期待どおりにユーザーが返されます。

_最初に報告されたのは 2024年6月27日（PT）です。_
