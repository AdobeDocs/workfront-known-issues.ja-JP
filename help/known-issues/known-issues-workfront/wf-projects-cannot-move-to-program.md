# プロジェクト：指定されたプログラムにプロジェクトを移動できない

ユーザーがプログラムにプロジェクトを移動しようとしても、プロジェクトが移動しません。 この問題は、プロジェクトが正常に移動されたというメッセージをユーザーが受け取った場合でも発生する可能性があります。

_最初に報告されたのは 2025年6月10日（PT）です。_
