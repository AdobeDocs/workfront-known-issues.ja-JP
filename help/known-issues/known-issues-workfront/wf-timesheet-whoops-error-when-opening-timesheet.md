# タイムシート：タイムシートを開く際にエラーが発生する

>[!NOTE]
>
>この問題は 2022年8月29日に修正されました。

ユーザーがタイムシートを開こうとすると、タイムシートが読み込まれず、次のエラーが表示されます。

「[!UICONTROL エラーが発生しました。[!DNL Workfront] にお問い合わせください。問題の原因を確認して修正させていただきます。]」

_最初に報告されたのは 2022年8月25日です。_
