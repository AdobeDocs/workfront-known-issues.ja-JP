# テンプレート：テンプレートをコピー中にエラーが発生する

>[!NOTE]
>
>この問題は、2024年2月1日（PT）に修正されました。

ユーザーが新規または既存のテンプレートをコピーしようとすると、テンプレートはコピーされず、次のエラーが表示されます。

「[!UICONTROL null にすることはできません]」

_最初に報告されたのは 2024年1月25日（PT）です。_
