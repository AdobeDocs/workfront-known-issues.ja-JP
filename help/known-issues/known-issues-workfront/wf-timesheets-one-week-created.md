# タイムシート：タイムシートが正しく生成されない

ユーザーがタイムシートを生成しようとして、生成が「1 週間ごと」に設定されている場合、1 週間のみが生成されます。

**回避策**

設定/ システム /診断/ タイムシートの生成から手動でタイムシートを生成します。

_最初に報告されたのは 2025年6月2日（PT）です。_
