# 更新：要求者ユーザーが更新に画像を追加できない

要求者ライセンスを持つユーザーが更新エリアを表示して画像を追加しようとすると、画像を追加するアイコンが表示されず、ユーザーは画像を追加するオプションを使用できません。

_最初に報告されたのは 2023年9月15日（PT）です。_
