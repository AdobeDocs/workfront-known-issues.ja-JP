# [!UICONTROL ホーム]：従来の[!UICONTROL ホームの作業リスト]で検索機能が機能しない

>[!NOTE]
>
>この問題は解決されました。

ユーザーが従来のホームエクスペリエンスで[!UICONTROL ホームの作業リスト]を表示し、リストを検索しようとしても、検索は無効になります。

_最初に報告されたのは 2023年8月18日（PT）です。_
