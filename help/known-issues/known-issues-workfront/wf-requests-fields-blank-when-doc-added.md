# リクエスト：ドキュメントがリクエストに追加されると、フィールドがクリアされます

>[!NOTE]
>
>この問題は、2023年11月11日に修正されました。

ユーザーがリクエストを作成し、フォームのフィールドに入力し、ドキュメントを追加または削除すると、フォームの一部のフィールドのデータが消去され、ユーザーはリクエストを送信する前に再入力する必要があります。

_最初に報告されたのは 2023年11月1日（PT）です。_
