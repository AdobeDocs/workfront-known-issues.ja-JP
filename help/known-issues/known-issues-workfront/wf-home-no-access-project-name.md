# ホーム：「アクセスなし」がプロジェクト名として表示される

ユーザーがホームで「マイタスク」ウィジェットを表示すると、プロジェクト名列に「アクセスなし」が表示されます。 この問題は、プロジェクト名が「自分の作業」ウィジェットに表示されている場合でも発生します。

_最初に報告されたのは 2024年8月12日（PT）です。_
