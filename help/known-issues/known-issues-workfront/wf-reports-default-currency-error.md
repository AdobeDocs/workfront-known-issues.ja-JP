# 報告書：デフォルトの通貨が USD の場合、報告書が読み込まれない

ユーザーが、デフォルトの通貨が USD であるレポートを表示しようとしても、レポートは読み込まれません。

**回避策**

デフォルトの通貨を「」に変更します。[!UICONTROL プロジェクトの既定通貨].」と入力します。

_最初に報告されたのは 2024年6月20日（PT）です。_
