# レポート：デフォルトの通貨が USD の場合、レポートが読み込まれない

>[!NOTE]
>
>この問題は 2024年6月27日（PT）に修正されました。

デフォルトの通貨が USD であるレポートをユーザーが表示しようとすると、レポートは読み込まれません。

**回避策**

デフォルトの通貨を「[!UICONTROL プロジェクトのデフォルトの通貨]」に変更します。

_最初に報告されたのは 2024年6月20日（PT）です。_
