# プルーフ：プルーフ名がブラウザータブに表示されない

ユーザーがプルーフを表示すると、プルーフの名前がブラウザータブに表示されません。代わりに、「Workfront」がブラウザータブに表示されます。

_最初に報告されたのは 2024年12月9日（PT）です。_
