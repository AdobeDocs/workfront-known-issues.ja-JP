# プルーフ：プルーフ名がブラウザータブに表示されない

ユーザーがプルーフを表示すると、ブラウザータブにプルーフの名前が表示されません。 代わりに、ブラウザータブには「Workfront」と表示されます。

_最初に報告されたのは 2024年12月9日（PT）です。_
