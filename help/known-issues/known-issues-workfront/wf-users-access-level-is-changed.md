# ユーザー：変更の記録がない場合にシステムによって変更されたアクセスレベル

ユーザのアクセスレベルは、システムによって予測不能に変更される可能性がある。 この場合、表示される更新はおこなわれず、変更は監査ログには表示されません。

_最初に報告されたのは 2023年2月18日（PT）です。_
