# ユーザー：変更記録なしで、システムによってアクセスレベルが変更される

>[!NOTE]
>
>このイシューは 2023年4月20日（PT）に修正されました。

ユーザーのアクセスレベルは、システムによって予期せず変更される場合があります。これが発生すると、表示されるアップデートはなく、変更は監査ログに表示されません。

_最初に報告されたのは 2023年2月18日（PT）です。_
