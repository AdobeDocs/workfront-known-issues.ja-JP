# タスク：大きなタスクリストが、リスト内の別の場所に予期せずジャンプする

ユーザーが大きなタスクリストを表示していて、タスクを更新したり、親タスクを展開したりすると、タスクリストがリスト内の別の場所に予期せずジャンプします。

_最初に報告されたのは、2026年1月22日（PT）です。_
