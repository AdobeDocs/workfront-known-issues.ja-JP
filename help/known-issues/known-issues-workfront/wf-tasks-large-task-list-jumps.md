# タスク：大きなタスクリストが予期せずリスト内の別の場所にジャンプする

ユーザーが大規模なタスクリストを表示しているときに、タスクを更新したり、親タスクを拡張したりすると、タスクリストが予期せずリスト内の別の場所にジャンプします。

_最初に報告されたのは2026年1月22日です。_
