# タスク：先行タスクのツールチップにタスク名が表示されません

ユーザーがタスクリストを表示し、先行タスクのアイコンにマウスポインターを置いて詳細を表示すると、表示されるツールチップには先行タスクの名前が表示されません。

_最初に報告されたのは 2024年3月5日（PT）です。_
