# タスク：先行タスクのツールヒントにタスク名が表示されない

>[!NOTE]
>
>このイシューは 2024年3月28日（PT）に修正されました。

ユーザーがタスクリストを表示し、詳細情報を表示するために先行タスクのアイコンにカーソルを合わせる際、表示されるツールヒントに先行タスクの名前が表示されません。

_最初に報告されたのは 2024年3月5日（PT）です。_
