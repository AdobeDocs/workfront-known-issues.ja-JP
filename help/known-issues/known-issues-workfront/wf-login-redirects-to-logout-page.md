# ログイン：ログアウトランディングページにリダイレクトされるログインページ

ユーザーが [!DNL Workfront]ログアウトする場合、ログインページではなく、ランディング先のページにリダイレクトされます。

**回避策**

を入力します。 [!DNL Workfront] URL を返し、再度ログインを試みます。
