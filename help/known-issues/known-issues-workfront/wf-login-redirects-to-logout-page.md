# ログイン：ログインページがログアウトランディングページにリダイレクトされる

ユーザーが [!DNL Workfront] にログインしようとすると、ログインページではなく、ログアウトした場合に表示されるページにリダイレクトされます。

**回避策**

[!DNL Workfront] URL を入力し、再度ログインを試みます。

_最初に報告されたのは 2023年10月5日（PT）です。_
