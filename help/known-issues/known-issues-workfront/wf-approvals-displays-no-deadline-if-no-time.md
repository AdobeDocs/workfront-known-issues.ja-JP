# 承認：期限に時間が含まれていない場合、承認は「期限なし」と表示される

承認期限に日付のみが含まれ、時間が含まれていない場合、承認ホームウィジェットの期限には「期限なし」と表示されます。

**回避策**

期限に時間を含めます。

_最初に報告されたのは 2025年8月21日（PT）です。_
