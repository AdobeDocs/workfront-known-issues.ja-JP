# リクエスト：リクエストの作成時に権限が継承されない

>[!NOTE]
>
>この問題は、2025年12月19日（PT）に修正されました。

ユーザーがリクエストを作成する際、そのリクエストに権限が継承されません。 つまり、ユーザーはリクエストにアクセスできない可能性があります。

_最初に報告されたのは、2025年11月20日（PT）です。_
