# リクエスト：リクエストの作成時に権限が継承されない

ユーザーがリクエストを作成しても、そのリクエストは権限を継承しません。 つまり、ユーザーはリクエストにアクセスできない可能性があります。

_最初に報告されたのは 2025年11月20日（PT）です。_
