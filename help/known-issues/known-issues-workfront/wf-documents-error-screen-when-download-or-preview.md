# ドキュメント：アセットのダウンロードまたはプレビュー時にエラー画面が表示される

>[!NOTE]
>
>この問題は 2024年6月27日（PT）に修正されました。

ユーザーが Workfront でアセットをダウンロードまたはプレビューしようとすると、エラー画面が表示されます。

_最初に報告されたのは 2024年2月2日（PT）です。_
