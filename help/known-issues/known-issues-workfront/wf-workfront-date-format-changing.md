# Workfront：アプリケーション全体で日付形式が変更される

<!--
>[!NOTE]
>
>This issue was fixed on August 30, 2024.
-->

アプリケーションでリストを使用して異なるページ間を移動する場合、日付の形式が MM/DD/YYYY から DD/MM/YYYY に散発的に変更されます（その逆も同様です）。

_最初に報告されたのは 2024年10月23日（PT）です。_
