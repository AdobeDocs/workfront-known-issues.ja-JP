# ボード：カードが一時的に元の列に戻る

>[!NOTE]
>
>このイシューは 2024年4月30日（PT）に修正されました。

ユーザーがボード上の別の列にカードを移動すると、カードは目的の列に移動する前に一時的に元の列に戻ります。

_最初に報告されたのは 2024年3月20日（PT）です。_
