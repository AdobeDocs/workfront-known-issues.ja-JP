# 通知:通知環境設定時のおっとエラー

ユーザーが環境設定表示 通知しようとすると、次のエラーが表示されます。

「[!UICONTROL エラーが発生しました。Workfront にお問い合わせください。問題の原因を確認して修正させていただきます。]」

この問題は、次のエリアで報告されています。

* ユーザー プロファイルの通知設定
* 「設定」のイベント通知領域

_最初に報告されたのは 2024年7月31日（PT）です。_
