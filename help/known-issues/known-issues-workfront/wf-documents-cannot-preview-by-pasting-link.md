# ドキュメント：プレビューリンクでドキュメントをプレビューできない

ユーザーは、次の方法でドキュメントプレビューを表示することができません。

* 右クリックしてリンクを新しいタブまたはウィンドウで開く
* 新しいタブまたはウィンドウにリンクをコピー＆ペーストする

この問題は、Adobe Unified Shell 上にあるアカウントで報告されています。

_最初に報告されたのは 2024年8月6日（PT）です。_
