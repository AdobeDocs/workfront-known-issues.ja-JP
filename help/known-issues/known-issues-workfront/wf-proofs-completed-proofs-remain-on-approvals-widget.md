# 配達確認：完了した配達確認は、 [!UICONTROL 承認待ち] widget

ユーザーが配達確認を決定した後、その配達確認は [!UICONTROL 承認待ち] ユーザーのウィジェット [!UICONTROL ホーム] ページを新しいホームエクスペリエンスに追加します。

_最初に報告されたのは 2023年9月21日（PT）です。_
