# カスタムフォーム：複数の値を選択するとドロップダウンが閉じる

>[!NOTE]
>
>この問題は 2024年7月25日（PT）に修正されました。

ユーザーがカスタムフォームフィールドで複数の値を選択しようとすると、最初の値が選択された後、ドロップダウンが閉じます。

この問題は、フィールドがカスタムフォームの表示ロジックに関連付けられている場合に発生します。

_最初に報告されたのは 2024年7月9日（PT）です。_
