# [!UICONTROL 設定]：監査ログを書き出せない

>[!NOTE]
>
>製品チームは現在、この問題の解決策を評価中です。この問題が解決されると、メンテナンス更新ではなく、製品のお知らせで通達されます。

ユーザーが大量の監査ログを書き出そうとしても、ログが書き出されません。書き出しが失敗したことは示されていません。

**回避策**

1 回の書き出しで書き出すログの数を減らします。

_最初に報告されたのは 2023年6月20日（PT）です。_
