# ホーム：自分の問題ウィジェットに、クローズした問題や完了した問題が表示されない

ホームの自分の問題ウィジェットに、クローズ済みまたは完了した問題が表示されません。 この問題は、これらの問題を表示するようにフィルターが設定されている場合でも発生する可能性があります。

_最初に報告されたのは 2024年5月31日（PT）です。_
