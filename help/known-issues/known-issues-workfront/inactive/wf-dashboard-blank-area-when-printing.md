# ダッシュボード：ダッシュボードを PDF に印刷する際に大きな余白ができる

<!--Article by request-->

ユーザーがダッシュボードを横向きで PDF に印刷すると、PDF の右側に大きな余白が表示されます。

**回避策**

縦向きモードで印刷すると、正しい余白が表示されます。

_最初に報告されたのは 2023年10月23日（PT）です。_
