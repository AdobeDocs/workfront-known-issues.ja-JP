# 承認：マイ承認ウィジェットに完了した承認が表示される

<!--

>[!NOTE]
>
>This issue was fixed on January 8, 2026.

-->

ユーザーがホームで「自分の承認」ウィジェットを表示すると、そのウィジェットには、既に承認されたプルーフが含まれます。

_最初に報告されたのは 2025年12月1日（PT）です。_
