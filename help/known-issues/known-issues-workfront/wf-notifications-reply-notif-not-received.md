# 通知：ユーザーにはコメントスレッドでの返信が通知されません

>[!NOTE]
>
>この問題は、2023年10月19日に修正されました。

ユーザーがコメントに返信した場合、その返信の通知を受け取る必要のある他のユーザーはその返信を受け取りません。 通知を受け取るユーザーと受け取らないユーザーがあります。

_最初に報告されたのは 2023年9月18日（PT）です。_
