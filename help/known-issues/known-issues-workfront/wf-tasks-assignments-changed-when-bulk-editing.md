# タスク：タスクを一括編集するとタスクの割り当てが変更される

>[!NOTE]
>
>この問題は、2022年11月10日に修正されました。


ユーザーがタスクのセットに関する任意のフィールドを一括編集すると、最初のタスクの割り当てがすべてのタスクに適用されます。これにより、前の割り当てが削除されます。

_最初に報告されたのは 2022年11月7日です。_
