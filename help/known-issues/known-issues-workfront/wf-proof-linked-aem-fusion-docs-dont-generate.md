# プルーフ：Fusion を通じてAEMからリンクされたドキュメントのプルーフを作成できない

Fusion を使用してAEMからWorkfrontにリンクされたプルーフにドキュメントを変換しようとすると、プロセスが停止し、プルーフが生成されません。

_最初に報告されたのは 2024年7月10日（PT）です。_
