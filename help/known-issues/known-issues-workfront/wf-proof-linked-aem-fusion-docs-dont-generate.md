# プルーフ：Fusion を通じて AEM からリンクされたドキュメントのプルーフを作成できない

Fusion を使用して AEM から Workfront にリンクされたドキュメントをプルーフに変換しようとすると、プロセスが停止し、プルーフが生成されません。

_最初に報告されたのは 2024年7月10日（PT）です。_
