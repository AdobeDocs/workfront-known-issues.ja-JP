# 配達確認：承認された配達確認が承認者の [!UICONTROL ホームの作業リスト]

<!--WF and WFP TOCs-->

>[!NOTE]
>
>この問題は 2023年8月17日に修正されました。

ユーザーが配達確認を承認すると、その配達確認の承認はユーザーの [!UICONTROL ホームの作業リスト]. 承認済みの配達確認は、 [!UICONTROL ホームの作業リスト].

_最初に報告されたのは 2023年7月21日（PT）です。_
