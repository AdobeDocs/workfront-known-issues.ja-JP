# ボード：複数のユーザーをフィルターすると、すべての結果が返されない

ボードの割り当てフィルターで複数のユーザーに対してユーザーがフィルターを実行した場合、予期した結果がすべて返されない。

**回避策**

詳細フィルターを使用して、各ユーザーのフィルターを作成し、OR ステートメントで結合します。

_最初に報告されたのは 2024年2月6日（PT）です。_
