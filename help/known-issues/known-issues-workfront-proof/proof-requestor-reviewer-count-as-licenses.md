# [!DNL Workfront Proof]:リクエスターとレビュー担当者は、組織のライセンス数にカウントされます

ユーザーがレビュー担当者またはリクエスト元として配達確認に追加されると、「[!UICONTROL 訪問者]「 」権限プロファイル。 [!DNL Workfront Proof] ライセンス。 ただし、ユーザーを追加すると、使用された [!DNL Workfront Proof] ライセンスが上がります。
