# [!DNL Workfront Proof]:リストが正しく並べ替えられていません

ユーザーがリストを表示したとき ( [!DNL Workfront Proof] 並べ替えが適用されている ( 例： [!UICONTROL 名前]) の場合、並べ替えは正確ではありません。 または、並べ替えを適用すると、リストは空白になります。

_最初に報告されたのは 2023年3月10日です。_
