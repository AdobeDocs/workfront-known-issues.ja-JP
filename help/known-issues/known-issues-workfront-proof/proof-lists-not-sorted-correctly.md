# [!DNL Workfront Proof]：リストが正しく並べ替えられていません

<!--Won't fix, valid issue-->

ユーザーが（[!UICONTROL 名前]などで）並べ替えが適用された [!DNL Workfront Proof] のリストを表示した場合、並べ替えは正確ではありません。または、並べ替えを適用すると、リストは空白になります。

**回避策**

フィルターを使用して、必要な情報を検索します。
