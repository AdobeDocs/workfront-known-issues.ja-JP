# [!DNL Workfront Proof]：フォルダービューにプルーフが表示されない

>[!NOTE]
>
>この問題は 2022年7月28日に修正されました。

ユーザーがプルーフをフォルダーに移動すると、不定な遅延が発生し、プルーフがフォルダーツリーに表示されません。

_最初に報告されたのは 2022年6月7日です。_
