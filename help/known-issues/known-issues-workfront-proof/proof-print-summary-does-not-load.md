# [!DNL Workfront Proof]: [!UICONTROL 印刷の概要] ページが読み込まれません

ユーザーが [!DNL Workfront Proof] 開こうとする [!UICONTROL 印刷の概要] ページに含まれている場合、ページは読み込み中のように見えますが、読み込まれません。

_最初に報告されたのは 2023年2月23日です。_
