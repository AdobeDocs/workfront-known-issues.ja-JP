# [!DNL Workfront Proof]:検索で結果が返されません

ユーザーが [!DNL Workfront Proof]の場合、検索結果は返されません。 検索によってユーザーが選択できる配達確認が提案される場合がありますが、ユーザーが検索を終了した場合は、これらの提案は結果に表示されません。

_最初に報告されたのは 2023年4月26日（PT）です。_
