# プルーフ：プルーフコメントでタグ付けされても、ユーザーにメールが届かない

>[!NOTE]
>
>この問題は再現できなかったので、クローズされました。

ユーザーがプルーフコメントで @name 形式を使用してタグ付けされても、コメントに関するメール通知がユーザーに届きません。

_最初に報告されたのは 2022年12月8日（PT）です。_
