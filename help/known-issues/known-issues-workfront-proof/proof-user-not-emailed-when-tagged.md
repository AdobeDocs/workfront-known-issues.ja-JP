# 配達確認：配達確認コメントでタグ付けされても、ユーザーが電子メールを受信しない

>[!NOTE]
>
>この問題は再現できなかったので、クローズされました。

配達確認コメントで@name形式を使用してユーザーがタグ付けされても、そのコメントに関する電子メール通知はユーザーに送信されません。

_最初に報告されたのは 2022年12月8日です。_
