# [!DNL Workfront Proof]：[!UICONTROL 新しいプルーフ] ページにテキストフィールドがない

>[!NOTE]
>
>この記事は 2022年8月18日に修正されました。

[!UICONTROL 新しいプルーフ]ページに、多くのテキストフィールド（フィールドラベル、ドロップダウンオプション、チェックボックス名を含む）が表示されません。
