# 内部使用のメタデータ

この [!DNL metadata].md ファイルには、ユーザーガイドに渡されるリポジトリレベルのメタデータが含まれています [!DNL TOC.md] ファイルをリポジトリに格納します。 任意のユーザガイドの metadata.md コンテンツを変更する場合は、任意の TOC.md ファイルで変更します。

詳しくは、 [メタデータ](https://experienceleague.adobe.com/docs/authoring-guide-exl/using/editing/user-guide-setup/metadata.html) の有効な値のリスト **ソリューション**, **製品**、および **type**.
